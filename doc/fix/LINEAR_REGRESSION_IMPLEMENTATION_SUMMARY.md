## 算法原理

### 线性回归法
对最近20个交易日的收盘价进行线性回归分析：

**判断标准**（三个条件都必须满足）：
1. **斜率 > 0**: 价格总体呈上升趋势
2. **p值 < 0.05**: 趋势显著，不是随机波动
3. **R² > 0.3**: 拟合度良好，趋势明显

**代码实现**：
```python
def _calculate_linear_regression(self, prices):
    # 反转为从旧到新
    prices_asc = prices[::-1]
    
    # X轴: 交易日序号
    X = np.arange(len(prices_asc))
    
    # 线性回归
    slope, intercept, r_value, p_value, std_err = stats.linregress(X, prices_asc)
    
    # 计算R²
    r_squared = r_value ** 2
    
    return slope, p_value, r_squared
```

## 优势对比

### 相比波峰波谷法
| 方面 | 波峰波谷法 | 线性回归法 |
|------|----------|----------|
| 识别准确性 | 容易识别错误 | 基于统计学，准确 |
| 滞后性 | 可能滞后 | 不滞后 |
| 参数数量 | 多（需要调整波峰波谷识别） | 少（3个参数） |
| 异常值处理 | 敏感 | 有一定容错能力 |
| 计算复杂度 | 高 | 低 |

### 相比均线法
| 方面 | 均线法 | 线性回归法 |
|------|--------|----------|
| 滞后性 | 可能滞后 | 不滞后 |
| 显著性检验 | 无 | 有（p值检验） |
| 拟合度评估 | 无 | 有（R²评估） |
| 准确性 | 中等 | 高 |
