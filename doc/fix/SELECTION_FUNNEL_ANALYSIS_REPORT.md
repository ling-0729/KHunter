# 选股漏斗分析报告 - 按顺序逐步验证

**报告日期**: 2026-03-21  
**分析方法**: 按照选股顺序逐步验证每一步的筛选效果  
**测试样本**: 500 只股票

---

## 执行摘要

### 选股流程漏斗

```
总股票数:                     500 (100.0%)
  ↓
步骤1 - 上升趋势:               15 (  3.0%) ✅
  ↓
步骤2 - 放量长阳线(10日):        0 (  0.0%) ❌ 瓶颈！
  ↓
步骤3 - 距离条件:                0 (  0.0%)
  ↓
步骤4 - 回调支撑:                0 (  0.0%)
  ↓
最终选中:                       0 (  0.0%)
```

### 关键发现

**瓶颈在步骤2**：虽然有 15 只股票满足上升趋势条件，但 0 只满足放量长阳线条件

---

## 详细分析

### 步骤1: 上升趋势检查

| 指标 | 数值 |
|------|------|
| 输入 | 500 只股票 |
| 输出 | 15 只股票 |
| 筛选率 | 97.0% |
| 保留率 | 3.0% |

**满足上升趋势的股票** (15 只):
- 000049, 000155, 000429, 000488, 000498, 000525, 000545, 000697, 000830, 000876
- ... 还有 5 只

**评价**: ✅ 正常，3% 的上升趋势股票比例合理

---

### 步骤2: 放量长阳线检查 (最近10个交易日内)

| 指标 | 数值 |
|------|------|
| 输入 | 15 只股票 (上升趋势) |
| 输出 | 0 只股票 |
| 筛选率 | 100.0% |
| 保留率 | 0.0% |

**问题**: ❌ 这 15 只上升趋势的股票中，没有一只在最近10个交易日内满足"涨幅 > 8% 且成交量 >= 2.0x"

**原因分析**:
1. 上升趋势的股票通常是缓慢上升，不一定有明显的放量长阳线
2. 涨幅 > 8% 的要求太高
3. 成交量 >= 2.0x 的要求也很高
4. 两个条件同时满足的概率极低

---

## 问题根源

### 参数设置过于严格

当前条件2的参数：
- 涨幅阈值: **8%** (太高)
- 成交量倍数: **2.0x** (太高)

这两个参数的组合导致：
- 即使是上升趋势的股票，也很难满足条件2
- 需要同时出现明显的放量和大幅上涨
- 这种情况在实际市场中很少见

### 为什么会这样

1. **上升趋势的特点**: 缓慢、稳定的上升，不一定有大幅波动
2. **放量长阳线的特点**: 需要明显的成交量增加和价格上涨
3. **两者的矛盾**: 上升趋势通常是温和的，而放量长阳线通常是突发的

---

## 解决方案

### 方案 A: 调整参数（推荐）

降低条件2的参数，使其更容易被满足：

| 参数 | 当前值 | 建议值 | 理由 |
|------|--------|--------|------|
| 涨幅阈值 | 8% | **5%** | 降低涨幅要求，更容易满足 |
| 成交量倍数 | 2.0x | **1.5x** | 降低成交量要求，更容易满足 |

**预期效果**:
- 步骤2 的通过率: 0% → 5-10%
- 最终选股数: 0 → 1-3 只

### 方案 B: 调整选股逻辑（可选）

改为"满足 3 个条件即可选股"而不是"必须满足 4 个条件"

### 方案 C: 分层选股（可选）

创建两个选股等级：
- 高质量: 满足所有 4 个条件（当前方案）
- 标准: 满足 3 个条件（放宽条件2）
