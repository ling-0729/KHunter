## 手续费计算规则

### 买入手续费
```
手续费 = max(成交金额 × 0.02%, 5 元)
```

**示例**:
- 成交金额 ¥54,000 → 手续费 = max(10.8, 5) = ¥10.8
- 成交金额 ¥1,000 → 手续费 = max(0.2, 5) = ¥5.0

### 卖出手续费
```
手续费 = max(成交金额 × 0.02%, 5 元)
```

**示例**:
- 成交金额 ¥54,700 → 手续费 = max(10.94, 5) = ¥10.94
- 成交金额 ¥1,000 → 手续费 = max(0.2, 5) = ¥5.0

### 印花税（仅卖出）
```
印花税 = 成交金额 × 0.1%
```

**示例**:
- 成交金额 ¥54,700 → 印花税 = ¥54.7

## 完整交易示例

### 买入示例

```
操作: 买入 5000 股平安银行 (000001)
价格: ¥10.80

计算过程:
1. 成交金额 = 5000 × 10.80 = ¥54,000
2. 手续费 = max(54000 × 0.02%, 5) = max(10.8, 5) = ¥10.8
3. 总成本 = 54000 + 10.8 = ¥54,010.8
4. 成本价 = 54010.8 / 5000 = ¥10.8022

结果:
- 成交金额: ¥54,000
- 手续费: ¥10.8
- 总成本: ¥54,010.8
- 成本价: ¥10.8022
```

### 卖出示例

```
操作: 卖出 5000 股平安银行 (000001)
价格: ¥10.94
成本价: ¥10.8022

计算过程:
1. 成交金额 = 5000 × 10.94 = ¥54,700
2. 手续费 = max(54700 × 0.02%, 5) = max(10.94, 5) = ¥10.94
3. 印花税 = 54700 × 0.1% = ¥54.7
4. 净收益 = 54700 - 10.94 - 54.7 = ¥54,634.36
5. 成本 = 5000 × 10.8022 = ¥54,011
6. 收益 = 54634.36 - 54011 = ¥623.36
7. 收益率 = 623.36 / 54011 × 100% = 1.15%

结果:
- 成交金额: ¥54,700
- 手续费: ¥10.94
- 印花税: ¥54.7
- 净收益: ¥54,634.36
- 收益: ¥623.36
- 收益率: 1.15%
```
