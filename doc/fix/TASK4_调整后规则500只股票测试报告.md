1. **条件1（深度下跌）**：15只股票满足
   - 这些股票在过去120个交易日内下跌超过45%
   - 比例较低（3.0%），说明市场中深度下跌的股票较少

2. **条件2（MACD底背离）**：91只股票满足
   - 这是三个条件中满足最多的
   - 比例为18.4%，说明有相当一部分股票出现了MACD底背离信号

3. **条件3（放量反弹）**：14只股票满足
   - 这是三个条件中满足最少的
   - 比例仅为2.8%，说明最近10个交易日内放量反弹的股票很少

4. **综合结果**：0只股票同时满足三个条件
   - 这与之前的分析结果一致
   - 反映了当前市场环境的特点：处于下跌通道，缺乏放量反弹

## 规则调整的影响

### 涨幅计算方式修正

**修正内容**：从 `(close - open) / open` 改为 `(close - prev_close) / prev_close`

**影响**：
- 更准确地反映股票相对于前一交易日的涨幅
- 符合股票分析的常见做法
- 可能会改变放量反弹的判断结果

### 最低点定义精确化

**修正内容**：最低点必须在最高点之后、在起涨点之前

**影响**：
- 确保最低点是在深度下跌过程中形成的底部
- 更严格的距离条件判断
- 可能会减少满足条件的股票数量
