## 参考算法分析

### 参考算法1: 均线斜率 + 价格位置（简单常用）
**原理**: 收盘价 > 20日均线 且 20日均线斜率 > 0

**优点**:
- 简单直观，易于理解
- 参数少，易于调整
- 计算快速

**缺点**:
- 可能滞后（均线是历史数据的平均）
- 对短期波动敏感
- 容易产生虚假信号

**评估**: ⭐⭐⭐ 适合快速判断，但准确性一般

---

### 参考算法2: 线性回归斜率 + 显著性（统计方法）
**原理**: 对20日收盘价进行线性回归，要求斜率 > 0 且 p-value < 0.05

**优点**:
- 统计学基础，科学严谨
- 考虑显著性，避免随机波动
- 能识别真实趋势

**缺点**:
- 计算复杂，需要统计库
- 对异常值敏感
- 参数调整困难

**评估**: ⭐⭐⭐⭐ 准确性高，但实现复杂

---

### 参考算法3: 波峰波谷识别（道氏理论简化）
**原理**: 识别20日内的局部高点和低点，要求最后一个低点 > 前一个低点，最后一个高点 > 前一个高点

**优点**:
- 符合技术分析理论
- 能识别真实的趋势转折
- 准确性高

**缺点**:
- 波峰波谷识别困难（容易识别错误）
- 需要足够的波峰波谷数量
- 对数据质量要求高

**评估**: ⭐⭐⭐⭐⭐ 准确性最高，但实现最复杂

## 当前实现分析

### 当前使用的方法
我们当前使用的是**参考算法3的完整版本**（道氏理论）：
- 识别所有波峰和波谷
- 检查所有波谷严格递增
- 检查所有波峰严格递增
- 检查每个波峰都高于相邻的波谷

### 当前问题
1. **波峰波谷识别不准确**: 使用相邻3个点判断，容易识别错误
2. **要求过于严格**: 要求所有波峰和波谷都严格递增，导致误判
3. **数据质量问题**: 某些股票数据有缺失或异常，导致波峰波谷识别失败

### 具体案例
- **000498**: 20日收益率 -3.44%（明显下跌），但被判定为上升趋势
  - 原因：虽然整体下跌，但最近几个波峰波谷满足递增条件
  - 问题：没有考虑整体趋势方向

## 最优方案建议

### 推荐方案：综合评分法（参考算法4的思想）

**核心思想**: 结合多个指标，综合评分，避免单一指标的片面性

**判断标准**（至少满足3个条件）:

1. **均线条件** (参考算法1)
   - 5日MA > 10日MA > 20日MA（多头排列）
   - 最新收盘价 > 5日MA
   - 权重: 40%

2. **价格位置条件** (参考算法2的思想)
   - 最新收盘价 > 20日平均价格
   - 最新收盘价 > 20日最低价 + (20日最高价 - 20日最低价) * 0.3
   - 权重: 30%

3. **趋势强度条件** (参考算法3的简化版)
   - 最近10日中，上升日 > 下降日（至少6个上升日）
   - 最近10日最高价 > 前10日最高价
   - 权重: 30%

**判断逻辑**:
```
综合得分 = 0
if 均线条件满足: 综合得分 += 1
if 价格位置条件满足: 综合得分 += 1
if 趋势强度条件满足: 综合得分 += 1

if 综合得分 >= 2: 判定为上升趋势
```

**优点**:
- ✅ 综合多个因素，避免单一指标的片面性
- ✅ 准确性高，能有效过滤虚假信号
- ✅ 参数少（3个），易于调整
- ✅ 计算简单，性能好
- ✅ 符合技术分析理论

**缺点**:
- 需要调整参数阈值

```python
def _check_uptrend(self, df) -> bool:
    """
    检查条件1：上升趋势（综合评分法）
    
    判断逻辑：
    - 综合考虑均线、价格位置、趋势强度
    - 至少满足2个条件即判定为上升趋势
    """
    if len(df) < 20:
        return False
    
    score = 0
    
    # 条件1: 均线条件
    if self._check_ma_condition(df):
        score += 1
    
    # 条件2: 价格位置条件
    if self._check_price_position_condition(df):
        score += 1
    
    # 条件3: 趋势强度条件
    if self._check_trend_strength_condition(df):
        score += 1
    
    return score >= 2
```

### 第二步: 添加辅助方法
- `_check_ma_condition()`: 检查均线条件
