
### 2. 时效检查逻辑修复

**文件**：`strategy/w_bottom_strategy.py`  
**方法**：`_check_neckline_break()`  
**行号**：第291-293行

**修改前**：
```python
if first_break_idx is not None and first_break_idx <= max_break_days:
    return first_break_idx
```

**修改后**：
