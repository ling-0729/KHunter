# W底策略优化总结

## 优化时间
2026年3月25日

## 优化背景

在分析300916朗特智能的选股结果时，发现该股票虽然满足W底形态的基本条件，但从L1到突破日的间隔较短（仅10天），这样的短期形态可能不够稳定。

为了提高选股质量，建议添加一个判断标准：**从L1到突破日，间隔不少于10个交易日**。

## 优化内容

### 1. 新增参数：min_pattern_days

**参数名**: `min_pattern_days`  
**默认值**: 10  
**含义**: 从L1（左底）到突破日的最小间隔交易日数  
**作用**: 过滤掉形态周期过短的股票，确保是真正的W底形态

### 2. 参数配置

在 `config/strategy_params.yaml` 中添加了新参数的配置：

```yaml
min_pattern_days:
  default: 10
  description: 从L1到突破日的最小间隔交易日数，用于过滤短期形态
  display_name: 最小形态天数
  group: 形态参数
  max: 30
  min: 5
  type: int
```

### 3. 代码实现

在 `strategy/w_bottom_strategy.py` 中：

**步骤1**: 在 `__init__` 方法中添加参数定义
```python
'min_pattern_days': 10,  # L1到突破日最小间隔（交易日）
```

**步骤2**: 在 `select_stocks` 方法中添加验证逻辑
```python
# 步骤3.5：验证形态时间间隔（L1到突破日至少10个交易日）
min_pattern_days = self.params['min_pattern_days']
pattern_interval = l1_idx - break_idx
if pattern_interval < min_pattern_days:
    return []
```

## 验证结果

### 测试1: 300916 朗特智能

| 参数值 | 结果 | 说明 |

| min_pattern_days=10 | 选入 | 形态间隔充足 |
| min_pattern_days=20 | 拒绝 | 形态间隔不足 |

**结论**: 300916的形态间隔在10-20天之间，使用默认参数(10)时被选入

## 优化效果

### 优点

1. **提高形态质量** - 过滤掉形态周期过短的股票
2. **增强稳定性** - 确保W底形态有足够的发展时间
3. **可配置性强** - 用户可根据需要调整参数
4. **向后兼容** - 默认参数为10，不影响现有选股结果

### 参数调整建议

- **激进策略** (min_pattern_days=5): 选股数量多，但形态稳定性相对较低
- **平衡策略** (min_pattern_days=10): 默认值，形态稳定性和选股数量的平衡
- **保守策略** (min_pattern_days=15-20): 选股数量少，但形态稳定性高
