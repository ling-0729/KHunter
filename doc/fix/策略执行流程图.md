# 策略执行流程图

## 整体执行流程

```
┌─────────────────────────────────────────────────────────────┐
│                    用户请求                                  │
│  GET /api/selection  或  POST /api/selection                │
└────────────────────┬────────────────────────────────────────┘
                     │
                     ▼
        ┌────────────────────────────┐
        │  解析请求参数              │
        │  - strategies: 策略列表    │
        │  - logic: or/and           │
        └────────────┬───────────────┘
                     │
                     ▼
        ┌────────────────────────────┐
        │  加载数据                  │
        │  - 股票代码列表            │
        │  - 股票名称映射            │
        │  - K线数据（CSV）          │
        └────────────┬───────────────┘
                     │
                     ▼
        ┌────────────────────────────┐
        │  过滤股票                  │
        │  - 数据充足（≥60条K线）    │
        │  - 可读取的股票            │
        └────────────┬───────────────┘
                     │
        ┌────────────┴────────────┐
        │                         │
        ▼                         ▼
    ┌─────────┐            ┌──────────┐
    │ OR逻辑  │            │ AND逻辑  │
    └────┬────┘            └────┬─────┘
         │                      │
         ▼                      ▼
```

```

│                    OR 逻辑执行                                │
│              （并集 - 任意策略选中即可）                      │
└────────────────────┬─────────────────────────────────────────┘
                     │
                     ▼
        ┌────────────────────────────┐
        │  遍历每个策略              │
        │  for strategy in strategies│
        └────────────┬───────────────┘
                     │
                     ▼

        │  遍历每只股票              │
        │  for stock in stocks       │
        └────────────┬───────────────┘
                     │
                     ▼
        ┌────────────────────────────┐
        │  执行策略分析              │
        │  result = strategy.        │
        │    analyze_stock(...)      │
        └────────────┬───────────────┘
                     │
        ┌────────────┴────────────┐
        │                         │
        ▼                         ▼
    ┌─────────┐            ┌──────────┐
    │ 有结果  │            │ 无结果   │
    └────┬────┘            └──────────┘
         │
         ▼
    ┌──────────────────┐
    │ 添加到结果列表   │
    │ signals.append() │
    └────┬─────────────┘
         │
         ▼
    ┌──────────────────────────────┐
    │ 所有股票都处理完成？         │
    └────┬──────────────┬──────────┘
         │ 是           │ 否

    ┌─────────┐    ┌──────────┐
    │ 下一策略 │    │ 继续处理 │

         ▼
```

## AND 逻辑详细流程

```
┌──────────────────────────────────────────────────────────────┐
│                    AND 逻辑执行                               │
│              （交集 - 所有策略都选中）                        │
└────────────────────┬─────────────────────────────────────────┘
                     │
                     ▼
        ┌────────────────────────────┐
        │  检查策略数量              │
        │  if len(strategies) > 1    │
        └────────────┬───────────────┘
                     │
        ┌────────────┴────────────┐
        │                         │
        ▼                         ▼
    ┌─────────┐            ┌──────────┐
    │ ≥2个    │            │ <2个     │
    └────┬────┘            └────┬─────┘
         │                      │
         ▼                      ▼
    ┌──────────────┐    ┌──────────────┐
    │ 继续执行     │    │ 转为OR逻辑   │
    │ AND逻辑      │    │ 返回所有结果 │
    └────┬─────────┘    └──────────────┘
         │
         ▼
    ┌────────────────────────────┐
    │  初始化交集集合            │
    │  all_signals = {}          │
    └────────────┬───────────────┘
         │
         ▼
    ┌────────────────────────────┐
    │  遍历每个策略              │
    │  for strategy in strategies│
    └────────────┬───────────────┘
         │
         ▼
    ┌────────────────────────────┐
    │  执行该策略的选股           │
    │  获得该策略的选中股票集合   │
    │  signals = [...]           │
    └────────────┬───────────────┘
         │
         ▼
    ┌────────────────────────────┐
    │  计算交集                  │
    │  if first_strategy:        │
    │    all_signals = signals   │
    │  else:                     │
    │    all_signals ∩= signals  │
    └────────────┬───────────────┘
         │
         ▼
    ┌────────────────────────────┐
    │  所有策略都处理完成？      │
    └────┬──────────────┬────────┘
         │ 是           │ 否
         ▼              ▼
    ┌──────────────┐  ┌──────────┐
    │ 返回交集结果 │  │ 继续处理 │
    │ {            │  └──────────┘
    │   "_inter    │
    │   section"   │
    │ }            │
    └──────────────┘
```

## 关键算法

### 交集计算（AND逻辑）
```python
# 初始化
all_signals = {}

# 遍历每个策略
for strategy_name in strategies:
    signals = execute_strategy(strategy_name)
    
    if not all_signals:
        # 第一个策略，直接赋值
        all_signals = {s['code']: s for s in signals}
    else:
        # 后续策略，计算交集
        all_signals = {
            code: s 
            for code, s in all_signals.items() 
            if any(sig['code'] == code for sig in signals)
        }

# 返回交集结果
return list(all_signals.values())
```

### 交集分析
```python
# 构建股票->策略映射
stock_strategies = {}
for strategy_name, signals in results.items():
    for signal in signals:
        code = signal['code']
        if code not in stock_strategies:
            stock_strategies[code] = {
                'code': code,
                'strategies': [],
                'count': 0
            }
        stock_strategies[code]['strategies'].append(strategy_name)
        stock_strategies[code]['count'] += 1

# 按策略数分组
by_count = {}
for code, data in stock_strategies.items():
    count = data['count']
    if count not in by_count:
        by_count[count] = []
    by_count[count].append(data)
```
