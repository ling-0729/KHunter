## 用户迁移指南

### 如果需要定时执行选股

#### Linux/Mac - 使用 crontab
```bash
# 编辑 crontab
crontab -e

# 添加以下行（每天15:05执行，仅工作日）
5 15 * * 1-5 cd /path/to/quant-csv && /usr/bin/python3 main.py run >> /var/log/quant-csv/run.log 2>&1
```

#### Windows - 使用任务计划程序
1. 打开"任务计划程序"
2. 创建基本任务
3. 设置触发器：每天15:05
4. 设置操作：运行程序 `python main.py run`
5. 设置工作目录为项目根目录

### 如果需要推送选股结果

#### 方案1：Web界面手动查看
- 访问 `http://localhost:5000`
- 执行选股
- 查看结果并手动保存

#### 方案2：自行集成第三方通知
- 在 `main.py` 中添加自定义通知逻辑
- 支持邮件、企业微信、飞书等

#### 方案3：使用Web API
- 调用 `/api/select` 获取选股结果
- 自行处理结果推送
