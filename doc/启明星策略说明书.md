# 启明星策略说明书

## 策略概述

启明星策略是一种基于三根K线形态识别的量化选股策略，用于捕捉股票底部反转信号。该策略通过识别经典的"启明星"形态，找出可能从下跌趋势转为上升趋势的股票。

### 策略特点

- **形态识别**：基于经典K线形态理论
- **底部反转**：专门识别底部反转信号
- **多重确认**：三根K线组合确认，降低误信号
- **量价配合**：结合成交量确认反转强度

## 选股条件

策略通过识别三根连续K线的特定组合来捕捉启明星形态：

### 形态定义

启明星形态由三根连续的K线组成，按时间顺序（从旧到新）如下：

#### 第一根K线：长阴线
- **特征**：收盘价 < 开盘价（阴线）
- **要求**：实体长度 > 阈值（默认0.01）
- **含义**：表示下跌趋势延续，市场恐慌

#### 第二根K线：小实体K线
- **特征**：开盘价和收盘价接近
- **要求**：实体长度 <= 第一根K线实体 × small_body_ratio（默认30%）
- **含义**：表示市场犹豫，下跌动能减弱

#### 第三根K线：长阳线
- **特征**：收盘价 > 开盘价（阳线）
- **要求**：
  - 实体长度 > 第一根K线实体 × long_candle_ratio（默认50%）
  - 收盘价 > 第一根K线的开盘价（突破）
  - 成交量 >= 第二根K线成交量 × volume_ratio（默认1.5倍）
- **含义**：表示反转确认，上升趋势开始

### 形态识别条件

**必须同时满足以下条件**：

1. **第一根K线是长阴线**
   - 收盘价 < 开盘价
   - 实体长度 >= 0.01（避免实体过小）

2. **第二根K线是小实体**
   - 实体长度 <= 第一根K线实体 × small_body_ratio（默认30%）
   - 可以是阳线或阴线

3. **第三根K线是长阳线**
   - 收盘价 > 开盘价
   - 实体长度 >= 0.01（避免实体过小）
   - 实体长度 >= 第一根K线实体 × long_candle_ratio（默认50%）
   - 收盘价 > 第一根K线的开盘价（突破关键位）
   - 成交量 >= 第二根K线成交量 × volume_ratio（默认1.5倍）

4. **时间范围**
   - 在lookback_days天内出现该形态（默认30天）

## 参数配置

### 参数说明

#### 形态参数
- `lookback_days`: 回溯天数
  - 默认值：30
  - 范围：3-60
  - 说明：在此天数内寻找启明星形态

- `small_body_ratio`: 小实体比例
  - 默认值：0.3
  - 范围：0.1-0.5
  - 说明：第二根K线实体与第一根K线实体的比例阈值

- `long_candle_ratio`: 长阳线比例
  - 默认值：0.5
  - 范围：0.3-1.0
  - 说明：第三根K线实体与第一根K线实体的最小比例

- `volume_ratio`: 成交量比例
  - 默认值：1.5
  - 范围：1.0-3.0
  - 说明：第三根K线成交量与第二根K线的比例

## 使用指南

### 基本使用

1. **启动系统**
   ```bash
   python web_server.py
   ```

2. **访问前端**
   - 打开浏览器，访问 http://localhost:5000

3. **执行选股**
   - 点击"执行选股"按钮
   - 在策略选择对话框中勾选"启明星策略"
   - 选择选股逻辑（OR 或 AND）
   - 点击"确认"执行选股

4. **查看结果**
   - 在"选股结果"页面查看选出的股票
   - 点击股票可查看详细信息
   - 查看启明星形态的详细信息（三根K线的日期、开盘价、收盘价）

### 参数调整

#### 激进策略（选股数量多）
- 增加 `lookback_days`（如60，延长回溯天数）
- 提高 `small_body_ratio`（如0.5，放宽小实体要求）
- 降低 `long_candle_ratio`（如0.3，放宽长阳线要求）
- 降低 `volume_ratio`（如1.0，放宽成交量要求）

#### 保守策略（选股数量少）
- 降低 `lookback_days`（如10，缩短回溯天数）
- 降低 `small_body_ratio`（如0.2，提高小实体要求）
- 提高 `long_candle_ratio`（如0.7，提高长阳线要求）
- 提高 `volume_ratio`（如2.0，提高成交量要求）

## 参数调整指南

### 调整思路

1. **选股数量过多**
   - 降低 `lookback_days`（缩短回溯天数）
   - 降低 `small_body_ratio`（提高小实体要求）
   - 提高 `long_candle_ratio`（提高长阳线要求）
   - 提高 `volume_ratio`（提高成交量要求）

2. **选股数量过少**
   - 提高 `lookback_days`（延长回溯天数）
   - 提高 `small_body_ratio`（放宽小实体要求）
   - 降低 `long_candle_ratio`（放宽长阳线要求）
   - 降低 `volume_ratio`（放宽成交量要求）

3. **选股质量不佳（形态不标准）**
   - 降低 `small_body_ratio`（确保第二根K线实体足够小）
   - 提高 `long_candle_ratio`（确保第三根K线实体足够大）
   - 提高 `volume_ratio`（确保成交量配合）
   - 降低 `lookback_days`（关注最近的形态）

### 参数优化建议

- 建议先使用默认参数运行，观察选股结果
- 根据实际选股数量和质量，逐步调整参数
- 每次调整一个参数，观察效果
- 避免过度优化，防止过拟合
- 关注形态的完整性，优先选择标准的启明星形态

## 常见问题

### Q1: 为什么没有选出任何股票？

**可能原因**：
- 参数设置过于严格
- 市场中没有符合条件的形态
- 数据不完整

**解决方案**：
- 提高 `small_body_ratio` 和 `long_candle_ratio`
- 降低 `volume_ratio`
- 增加 `lookback_days`
- 检查数据是否完整

### Q2: 选出的股票太多，如何筛选？

**解决方案**：
- 降低 `lookback_days`
- 降低 `small_body_ratio`
- 提高 `long_candle_ratio`
- 提高 `volume_ratio`
- 结合其他指标进行二次筛选

### Q3: 策略的准确率如何？

**说明**：
- 策略基于经典K线形态理论，准确率取决于市场环境
- 启明星形态是经典的底部反转信号，具有较高的参考价值
- 建议结合基本面分析进行综合判断
- 建议设置止损点，控制风险
- 成交量配合的形态通常更可靠

### Q4: 如何理解启明星形态？

**说明**：
- **第一根长阴线**：表示下跌趋势延续，市场恐慌
- **第二根小实体**：表示市场犹豫，下跌动能减弱，可能形成底部
- **第三根长阳线**：表示反转确认，上升趋势开始
- **突破第一根开盘价**：表示突破关键阻力位，反转确认

### Q5: 启明星形态与早晨之星的区别？

**说明**：
- **启明星**：本策略使用的名称，指三根K线的底部反转形态
- **早晨之星**：西方技术分析中的经典形态，与启明星类似
- 本策略实现的启明星形态与早晨之星基本一致
- 关键区别在于参数设置和判断标准

## 技术细节

### 数据要求
- 最少3个交易日的日线数据（用于识别形态）
- 建议至少30个交易日数据（用于计算KDJ和趋势线）
- 包含：开盘价、最高价、最低价、收盘价、成交量

### 计算流程

1. **数据准备**
   - 加载股票日线数据
   - 验证数据完整性和有效性

2. **指标计算**
   - 计算K线实体大小（|收盘价 - 开盘价|）
   - 计算K线方向（阳线=1，阴线=-1）
   - 计算成交量比例
   - 计算KDJ指标（K、D、J值）
   - 计算趋势线（短期趋势、牛熊线）

3. **形态识别**
   - 在lookback_days范围内遍历三根K线的组合
   - 检查是否满足启明星形态的所有条件
   - 记录满足条件的形态信息

4. **结果过滤**
   - 过滤 ST、*ST、退市股票
   - 过滤数据异常的股票
   - 生成选股信号和形态详情

5. **返回结果**
   - 返回符合条件的股票列表
   - 包含启明星形态的详细信息
