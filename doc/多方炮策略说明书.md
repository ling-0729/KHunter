# 多方炮策略说明书

## 策略概述

多方炮策略是一种识别"两阳夹一阴"K线组合形态的量化选股策略。该策略通过识别多方短暂回调后再次发力的经典形态，找出具有上涨延续或反转潜力的股票。

### 策略特点

- **形态经典**：两阳夹一阴是经典的多头形态，市场认可度高
- **信号明确**：K线形态清晰，易于识别
- **多重验证**：结合形态、成交量、趋势多重条件确认
- **风险可控**：止损点明确（第二根K线最低点）

## 选股条件

策略通过以下四个关键条件的组合来识别多方炮形态：

### 条件1：第一根K线（中阳线或大阳线）

识别第一根阳线，代表多方第一次进攻。

**判断方法**：
- 收盘价 > 开盘价（阳线）
- 涨幅 >= 3%（中阳线）
- 实体大小适中

**参数**：
- `first_candle_rise`: 第一根阳线涨幅阈值（默认0.03 = 3%）

### 条件2：第二根K线（小阴线或十字星）

识别第二根阴线，代表空方反击但力度有限。

**判断方法**：
- 收盘价 < 开盘价（阴线）
- 实体大小 <= 第一根阳线实体的50%
- 回调幅度 <= 第一根阳线涨幅的50%
- 成交量 <= 第一根阳线成交量的80%（缩量）

**参数**：
- `second_candle_body_ratio`: 第二根阴线实体比例（默认0.5）
- `second_candle_fallback_ratio`: 第二根阴线回调比例（默认0.5）
- `second_volume_shrink_ratio`: 第二根阴线缩量比例（默认0.8）

### 条件3：第三根K线（阳线）

识别第三根阳线，代表多方再次发力并突破前高。

**判断方法**：
- 收盘价 > 开盘价（阳线）
- 涨幅 >= 3%（中阳线）
- 收盘价 > 第一根K线的收盘价（突破确认）
- 成交量 >= 第一根阳线成交量的120%（放量）
- 成交量 >= 1.5倍5日均量

**参数**：
- `third_candle_rise`: 第三根阳线涨幅阈值（默认0.03 = 3%）
- `third_candle_breakthrough`: 突破前高（默认true）
- `third_volume_expand_ratio`: 第三根阳线放量比例（默认1.2）
- `third_volume_ma_ratio`: 第三根阳线放量倍数（默认1.5）
- `volume_ma_period`: 成交量周期（默认5）

### 条件4：趋势过滤（可选）

通过趋势指标过滤，提高选股质量。

**判断方法**：
- 均线过滤（可选）：股价 > MA20
- MACD过滤（可选）：MACD > 0
- KDJ过滤（可选）：J < 80（避免超买）

**参数**：
- `enable_ma_filter`: 启用均线过滤（默认true）
- `ma_period`: 均线周期（默认20）
- `enable_macd_filter`: 启用MACD过滤（默认false）
- `macd_above_zero`: MACD大于0（默认true）
- `enable_kdj_filter`: 启用KDJ过滤（默认false）
- `kdj_j_max`: KDJ的J值上限（默认80）

## 参数配置

### 参数分组

#### 形态参数
- `first_candle_rise`: 第一根阳线涨幅(%)
  - 默认值：0.03
  - 范围：0.01-0.10
  - 说明：第一根阳线涨幅需要超过此值

- `second_candle_body_ratio`: 第二根阴线实体比例
  - 默认值：0.5
  - 范围：0.3-0.8
  - 说明：第二根阴线实体占第一根阳线实体的比例

- `second_candle_fallback_ratio`: 第二根阴线回调比例
  - 默认值：0.5
  - 范围：0.3-0.8
  - 说明：第二根阴线回调占第一根阳线涨幅的比例

- `third_candle_rise`: 第三根阳线涨幅(%)
  - 默认值：0.03
  - 范围：0.01-0.10
  - 说明：第三根阳线涨幅需要超过此值

- `third_candle_breakthrough`: 突破前高
  - 默认值：true
  - 类型：布尔值
  - 说明：第三根阳线收盘价是否需要突破第一根阳线收盘价

#### 成交量参数
- `second_volume_shrink_ratio`: 第二根阴线缩量比例
  - 默认值：0.8
  - 范围：0.5-0.9
  - 说明：第二根阴线成交量占第一根阳线成交量的比例

- `third_volume_expand_ratio`: 第三根阳线放量比例
  - 默认值：1.2
  - 范围：1.0-2.0
  - 说明：第三根阳线成交量占第一根阳线成交量的比例

- `third_volume_ma_ratio`: 第三根阳线放量倍数
  - 默认值：1.5
  - 范围：1.0-3.0
  - 说明：第三根阳线成交量占均量的比例

- `volume_ma_period`: 成交量周期
  - 默认值：5
  - 范围：3-10
  - 说明：成交量均线计算周期

#### 趋势过滤
- `enable_ma_filter`: 启用均线过滤
  - 默认值：true
  - 类型：布尔值
  - 说明：是否启用均线过滤条件

- `ma_period`: 均线周期
  - 默认值：20
  - 范围：5-60
  - 说明：均线过滤使用的均线周期

- `enable_macd_filter`: 启用MACD过滤
  - 默认值：false
  - 类型：布尔值
  - 说明：是否启用MACD过滤条件

- `macd_above_zero`: MACD大于0
  - 默认值：true
  - 类型：布尔值
  - 说明：MACD是否需要大于0

- `enable_kdj_filter`: 启用KDJ过滤
  - 默认值：false
  - 类型：布尔值
  - 说明：是否启用KDJ过滤条件

- `kdj_j_max`: KDJ的J值上限
  - 默认值：80
  - 范围：70-100
  - 说明：KDJ的J值不能超过此值

#### 其他参数
- `lookback_days`: 回溯天数
  - 默认值：8
  - 范围：5-30
  - 说明：在此范围内寻找多方炮形态

- `min_market_cap`: 最小市值(亿)
  - 默认值：20
  - 范围：10-100
  - 说明：股票总市值的最小值（亿元）

- `max_market_cap`: 最大市值(亿)
  - 默认值：1000
  - 范围：100-5000
  - 说明：股票总市值的最大值（亿元）

## 使用指南

### 基本使用

1. **启动系统**
   ```bash
   python web_server.py
   ```

2. **访问前端**
   - 打开浏览器，访问 http://localhost:5000

3. **执行选股**
   - 点击"执行选股"按钮
   - 在策略选择对话框中勾选"多方炮策略"
   - 选择选股逻辑（OR 或 AND）
   - 点击"确认"执行选股

4. **查看结果**
   - 在"选股结果"页面查看选出的股票
   - 点击股票可查看详细K线图

### 参数调整

#### 激进策略（选股数量多）
- 降低 `first_candle_rise`（如0.01）
- 降低 `third_candle_rise`（如0.01）
- 增加 `second_volume_shrink_ratio`（如0.9）
- 降低 `third_volume_expand_ratio`（如1.0）
- 禁用趋势过滤（`enable_ma_filter`设为false）

#### 保守策略（选股数量少）
- 提高 `first_candle_rise`（如0.05）
- 提高 `third_candle_rise`（如0.05）
- 降低 `second_volume_shrink_ratio`（如0.7）
- 提高 `third_volume_expand_ratio`（如1.5）
- 启用趋势过滤（`enable_ma_filter`、`enable_macd_filter`设为true）

## 参数调整指南

### 调整思路

1. **选股数量过多**
   - 提高 `first_candle_rise`（涨幅要求更高）
   - 提高 `third_candle_rise`（涨幅要求更高）
   - 降低 `second_volume_shrink_ratio`（缩量要求更严格）
   - 提高 `third_volume_expand_ratio`（放量要求更高）
   - 启用趋势过滤（增加过滤条件）

2. **选股数量过少**
   - 降低 `first_candle_rise`（涨幅要求降低）
   - 降低 `third_candle_rise`（涨幅要求降低）
   - 增加 `second_volume_shrink_ratio`（缩量要求放松）
   - 降低 `third_volume_expand_ratio`（放量要求降低）
   - 禁用趋势过滤（减少过滤条件）

3. **选股质量不佳**
   - 启用 `enable_ma_filter`（确保上升趋势）
   - 启用 `enable_macd_filter`（确保MACD向上）
   - 启用 `enable_kdj_filter`（避免超买）
   - 提高 `third_volume_ma_ratio`（确保放量明显）
   - 降低 `max_market_cap`（排除大盘股）

### 参数优化建议

- 建议先使用默认参数运行，观察选股结果
- 根据实际选股数量和质量，逐步调整参数
- 每次调整一个参数，观察效果
- 避免过度优化，防止过拟合

## 形态分类

根据第一根和第三根K线的涨幅，将多方炮形态分为三类：

### 强势多方炮
- **涨幅**：>= 7%
- **特征**：多方力量强劲，上涨空间大
- **风险**：可能已处于高位
- **建议**：适合激进投资者，注意止损

### 标准多方炮
- **涨幅**：3% - 7%
- **特征**：多方力量适中，上涨概率高
- **风险**：风险收益比合理
- **建议**：适合大多数投资者

### 弱势多方炮
- **涨幅**：1% - 3%
- **特征**：多方力量较弱
- **风险**：可能是上涨中继形态
- **建议**：需结合其他指标判断

## 常见问题

### Q1: 为什么没有选出任何股票？

**可能原因**：
- 参数设置过于严格
- 市场中没有符合条件的形态
- 数据不完整

**解决方案**：
- 降低 `first_candle_rise` 和 `third_candle_rise`
- 增加 `second_volume_shrink_ratio`
- 降低 `third_volume_expand_ratio`
- 禁用趋势过滤
- 检查数据是否完整

### Q2: 选出的股票太多，如何筛选？

**解决方案**：
- 提高 `first_candle_rise` 和 `third_candle_rise`
- 降低 `second_volume_shrink_ratio`
- 提高 `third_volume_expand_ratio`
- 启用趋势过滤（`enable_ma_filter`、`enable_macd_filter`）
- 降低 `max_market_cap`（排除大盘股）
- 结合其他指标进行二次筛选

### Q3: 策略的准确率如何？

**说明**：
- 多方炮是经典形态，市场认可度高
- 但准确率取决于市场环境和个股情况
- 建议结合基本面分析进行综合判断
- 建议设置止损点，控制风险

### Q4: 如何设置止损点？

**建议**：
- 在第二根K线最低点设置止损
- 止损幅度通常在3%-5%
- 严格执行止损，避免深度套牢

## 技术细节

### 数据要求

- 最少10个交易日的日线数据
- 包含：开盘价、最高价、最低价、收盘价、成交量

### 计算流程

1. 计算指标（KDJ、MACD、均线、成交量均线等）
2. 计算K线特征（实体大小、涨幅、方向等）
3. 在回溯天数内寻找三根连续K线
4. 检查条件1：第一根K线是中阳线
5. 检查条件2：第二根K线是小阴线且缩量
6. 检查条件3：第三根K线是阳线且放量突破
7. 检查条件4：趋势过滤（可选）
8. 过滤 ST、*ST、退市股票
9. 过滤市值不符合的股票
10. 返回选股结果

### 性能指标

- 处理500只股票耗时 < 30秒
- 内存占用 < 500MB
- 支持实时更新

## 注意事项

### 适用场景

1. **上涨趋势中的回调**：多方炮形态在上涨趋势中出现，成功率较高
2. **底部反转**：在长期下跌后出现多方炮，可能预示反转
3. **突破整理平台**：突破横盘整理平台时出现多方炮，信号更可靠

### 不适用场景

1. **下跌趋势中**：在下跌趋势中出现多方炮，成功率较低
2. **高位震荡**：在高位震荡中出现多方炮，可能是诱多
3. **重大利空**：在重大利空消息影响下，多方炮可能失效

### 风险控制

1. **设置止损**：建议在第二根K线最低点设置止损
2. **分批建仓**：不要一次性满仓，可以分批建仓
3. **结合其他指标**：结合MACD、KDJ、均线等指标提高成功率
4. **注意成交量**：成交量是多方炮形态的重要验证指标
