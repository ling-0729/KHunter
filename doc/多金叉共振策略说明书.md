# 多金叉共振策略说明书

## 策略概述

多金叉共振策略是一种识别多重技术指标同时发出买入信号的量化选股策略。当均线金叉、KDJ金叉、MACD金叉三者同时发生或相隔不到3天时，形成强烈的上涨共振信号，预示股价可能继续上涨，是重要的买入信号。

### 策略特点

- **多重确认**：三个技术指标同时发出买入信号，降低误判
- **信号强度**：共振信号比单一金叉信号更强
- **趋势反转**：专门捕捉上涨趋势的开始
- **机会把握**：帮助投资者及时抓住上涨机会

## 选股条件

策略通过以下五个关键条件的组合来识别多金叉共振形态：

### 条件1：均线金叉

识别短期均线上穿长期均线的金叉信号。

**判断方法**：
- 计算短期均线（默认5日MA5）和长期均线（默认20日MA20）
- 当日：MA5 > MA20
- 前一日：MA5 <= MA20
- 满足上述条件即为均线金叉

**参数**：
- `ma_short_period`: 短期均线周期（默认5）
- `ma_long_period`: 长期均线周期（默认20）

### 条件2：KDJ金叉

识别KDJ指标中K线上穿D线的金叉信号。

**判断方法**：
- 计算KDJ指标（N=9, M1=3, M2=3参数）
- 当日：K > D
- 前一日：K <= D
- 满足上述条件即为KDJ金叉

**参数**：
- `kdj_n`: KDJ的N参数（默认9）
- `kdj_m1`: KDJ的M1参数（默认3）
- `kdj_m2`: KDJ的M2参数（默认3）

### 条件3：MACD金叉

识别MACD指标中DIF线上穿DEA线的金叉信号。

**判断方法**：
- 计算MACD指标（短期EMA=12，长期EMA=26，信号线EMA=9）
- 当日：DIF > DEA
- 前一日：DIF <= DEA
- 满足上述条件即为MACD金叉

**参数**：
- `macd_short`: MACD短期EMA周期（默认12）
- `macd_long`: MACD长期EMA周期（默认26）
- `macd_signal`: MACD信号线EMA周期（默认9）

### 条件4：共振确认

确认三个金叉信号在时间窗口内同时发生或相隔不远。

**判断方法**：
- 找到三个金叉信号的发生日期
- 计算最早日期和最晚日期的时间差
- 时间差 <= 3天即满足共振条件

**参数**：
- `resonance_days`: 共振时间窗口（默认3天）
- `lookback_days`: 回溯天数（默认10天）

### 条件5：价格确认

通过价格位置确认上涨趋势的有效性。

**判断方法**：
- 收盘价 > MA5
- 收盘价 > MA20
- 收盘价 > 短期趋势线
- 收盘价 > 多空线
- 满足上述所有条件即通过价格确认

## 参数配置

### 参数分组

#### 均线参数
- `ma_short_period`: 短期均线周期
  - 默认值：5
  - 范围：3-10
  - 说明：短期均线周期，用于判断短期趋势

- `ma_long_period`: 长期均线周期
  - 默认值：20
  - 范围：10-60
  - 说明：长期均线周期，用于判断长期趋势

#### KDJ参数
- `kdj_n`: KDJ的N参数
  - 默认值：9
  - 范围：5-20
  - 说明：KDJ指标的RSV周期

- `kdj_m1`: KDJ的M1参数
  - 默认值：3
  - 范围：1-5
  - 说明：KDJ指标的K值平滑周期

- `kdj_m2`: KDJ的M2参数
  - 默认值：3
  - 范围：1-5
  - 说明：KDJ指标的D值平滑周期

#### MACD参数
- `macd_short`: MACD短期EMA周期
  - 默认值：12
  - 范围：5-20
  - 说明：MACD指标的短期EMA周期

- `macd_long`: MACD长期EMA周期
  - 默认值：26
  - 范围：10-40
  - 说明：MACD指标的长期EMA周期

- `macd_signal`: MACD信号线EMA周期
  - 默认值：9
  - 范围：5-20
  - 说明：MACD指标的信号线EMA周期

#### 共振参数
- `resonance_days`: 共振时间窗口
  - 默认值：3
  - 范围：1-5
  - 说明：三个金叉信号的最大时间差（天）

- `lookback_days`: 回溯天数
  - 默认值：10
  - 范围：5-20
  - 说明：在此天数内寻找金叉信号

#### 其他参数
- `min_market_cap`: 最小市值(亿)
  - 默认值：20
  - 范围：10-100
  - 说明：股票总市值的最小值（亿元）

- `max_market_cap`: 最大市值(亿)
  - 默认值：1000
  - 范围：100-5000
  - 说明：股票总市值的最大值（亿元）

## 使用指南

### 基本使用

1. **启动系统**
   ```bash
   python web_server.py
   ```

2. **访问前端**
   - 打开浏览器，访问 http://localhost:5000

3. **执行选股**
   - 点击"执行选股"按钮
   - 在策略选择对话框中勾选"多金叉共振策略"
   - 选择选股逻辑（OR 或 AND）
   - 点击"确认"执行选股

4. **查看结果**
   - 在"选股结果"页面查看选出的股票
   - 点击股票可查看详细K线图

### 参数调整

#### 激进策略（选股数量多）
- 增加 `resonance_days`（如5天）
- 增加 `lookback_days`（如15天）
- 缩短 `ma_short_period`（如3日）
- 缩短 `ma_long_period`（如10日）

#### 保守策略（选股数量少）
- 降低 `resonance_days`（如1天）
- 降低 `lookback_days`（如5天）
- 延长 `ma_short_period`（如10日）
- 延长 `ma_long_period`（如60日）

## 参数调整指南

### 调整思路

1. **选股数量过多**
   - 降低 `resonance_days`（共振要求更严格）
   - 降低 `lookback_days`（回溯天数减少）
   - 延长 `ma_short_period`（短期均线周期增加）
   - 延长 `ma_long_period`（长期均线周期增加）

2. **选股数量过少**
   - 增加 `resonance_days`（放宽共振要求）
   - 增加 `lookback_days`（回溯天数增加）
   - 缩短 `ma_short_period`（短期均线周期减少）
   - 缩短 `ma_long_period`（长期均线周期减少）

3. **选股质量不佳**
   - 调整KDJ参数（`kdj_n`、`kdj_m1`、`kdj_m2`）
   - 调整MACD参数（`macd_short`、`macd_long`、`macd_signal`）
   - 调整市值范围（`min_market_cap`、`max_market_cap`）

### 参数优化建议

- 建议先使用默认参数运行，观察选股结果
- 根据实际选股数量和质量，逐步调整参数
- 每次调整一个参数，观察效果
- 避免过度优化，防止过拟合

## 信号强度分类

根据三个金叉信号的时间差，将多金叉共振信号分为三类：

### 强共振信号
- **时间差**：同一天发生
- **特征**：三个指标在同一天发出金叉信号
- **信号强度**：最强，上涨概率最高
- **建议**：强烈建议买入

### 标准共振信号
- **时间差**：1-2天内
- **特征**：三个指标在相邻交易日发出金叉信号
- **信号强度**：较强，上涨概率较高
- **建议**：建议买入

### 弱共振信号
- **时间差**：3天内
- **特征**：三个指标在3天内发出金叉信号
- **信号强度**：中等，需要结合其他指标判断
- **建议**：谨慎买入，结合其他因素判断

## 常见问题

### Q1: 为什么没有选出任何股票？

**可能原因**：
- 参数设置过于严格
- 市场处于下跌阶段，金叉信号较少
- 数据不完整

**解决方案**：
- 增加 `resonance_days`（放宽共振要求）
- 增加 `lookback_days`（增加回溯天数）
- 缩短 `ma_short_period` 和 `ma_long_period`
- 检查数据是否完整

### Q2: 选出的股票太多，如何筛选？

**解决方案**：
- 降低 `resonance_days`（共振要求更严格）
- 降低 `lookback_days`（减少回溯天数）
- 延长 `ma_short_period` 和 `ma_long_period`
- 调整KDJ和MACD参数
- 结合其他指标进行二次筛选

### Q3: 策略的准确率如何？

**说明**：
- 多金叉共振是多重技术指标确认的信号
- 准确率取决于市场环境和个股情况
- 历史数据显示共振信号比单一信号更可靠
- 建议结合基本面分析进行综合判断

### Q4: 如何使用该策略进行买入操作？

**建议**：
- 当股票出现多金叉共振信号时，考虑建仓或加仓
- 设置止损点，控制风险
- 结合其他技术指标确认信号
- 关注市场整体趋势，顺势而为

## 技术细节

### 数据要求

- 最少20个交易日的日线数据
- 包含：开盘价、最高价、最低价、收盘价、成交量

### 计算流程

1. 计算指标（KDJ、MACD、均线、成交量均线等）
2. 在回溯天数内查找均线金叉日期
3. 在回溯天数内查找KDJ金叉日期
4. 在回溯天数内查找MACD金叉日期
5. 计算三个金叉信号的时间差
6. 检查共振条件（时间差 <= resonance_days）
7. 检查价格确认条件
8. 过滤 ST、*ST、退市股票
9. 过滤市值不符合的股票
10. 返回选股结果

### 性能指标

- 处理500只股票耗时 < 30秒
- 内存占用 < 500MB
- 支持实时更新

## 注意事项

### 适用场景

1. **下跌趋势中的反弹**：在下跌趋势中出现多金叉共振，可能是趋势反转信号
2. **底部区域**：在底部区域出现多金叉共振，可能是上涨开始
3. **上涨趋势确认**：在上涨趋势中出现多金叉共振，确认上涨趋势

### 不适用场景

1. **高位区域**：在高位区域出现多金叉共振，可能是诱多
2. **横盘整理**：在横盘整理中出现多金叉共振，信号可能失效
3. **重大利空**：在重大利空消息影响下，多金叉共振可能失效

### 风险控制

1. **结合趋势判断**：多金叉共振在上涨趋势中更可靠
2. **关注成交量**：成交量放大时信号更可靠
3. **结合其他指标**：结合RSI、布林带等指标提高成功率
4. **注意市场环境**：在熊市中，多金叉共振可能是短暂反弹
