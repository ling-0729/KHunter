# 底部趋势拐点策略说明书

## 策略概述

底部趋势拐点策略是一种识别股票在深度下跌后出现反转拐点的量化选股策略。该策略通过深度下跌、MACD底背离、放量反弹三个关键条件的组合，找出处于底部反转阶段的股票。

### 策略特点

- **底部反转**：专门识别深度下跌后的反转机会
- **多指标组合**：结合价格、技术指标、成交量三个维度
- **严格筛选**：三个条件必须同时满足，确保选股质量
- **风险控制**：通过底背离确认下跌动能衰竭

---

## 选股规则详解

### 规则1：深度下跌（条件1）

**规则描述**: 股票从高点到当前价格的下跌幅度超过 45%

**详细说明**:
- **时间范围**: 回溯 120 个交易日（约半年）
- **高点定义**: 在 120 个交易日内出现的最高价
- **关键要求**: 高点必须在时间上更早，最低点必须在时间上更近（形成真正的下跌）
- **计算公式**: 下跌幅度 = (最高价 - 最低价) / 最高价
- **判断条件**: 下跌幅度 > 45%

**实际含义**:
- 股票从半年内的高点下跌了超过 45%
- 这表明股票处于深度下跌状态
- 是寻找底部反转的前提条件

**参数**:
- `decline_threshold`: 0.45 (45%)
- `lookback_days`: 120 (交易日)

---

### 规则2：MACD 底背离（条件2）

**规则描述**: 股票价格创新低，但 MACD 指标没有继续创新低

**详细说明**:
- **时间范围**: 最近 20 个交易日
- **判断逻辑**: 
  - 当前价格 < 前 20 天平均价格（价格在下降）
  - 当前 MACD > 前 20 天平均 MACD（MACD 没有继续下降）
- **两个条件都必须满足**

**实际含义**:
- 价格继续下跌，但动能（MACD）开始减弱
- 这是底部反转的重要信号
- 表明下跌动能已经衰竭，反转可能即将发生

**MACD 计算方式**:
- DIF = 12 日 EMA - 26 日 EMA
- DEA = DIF 的 9 日 EMA
- MACD = DIF - DEA

**参数**:
- `macd_divergence_days`: 20 (交易日)

---

### 规则3：放量反弹（条件3）

**规则描述**: 在最近 10 个交易日内出现放量反弹

**详细说明**:

#### 3.1 涨幅条件
- **要求**: 当日涨幅 > 8% 或涨停（≥ 9.5%）
- **计算**: 涨幅 = (当日收盘价 - 前一日收盘价) / 前一日收盘价
- **说明**: 直接从历史数据中取前一日收盘价，计算当日相对于前一日的涨幅

#### 3.2 成交量条件
- **要求**: 当日成交量 ≥ 2.5 倍前 10 日平均成交量
- **计算**: 成交量比 = 当日成交量 / 前 10 日平均成交量
- **说明**: 前 10 日平均成交量不包括当日

#### 3.3 距离条件
- **要求**: 起涨点距离最低点不超过 15%
- **起涨点定义**: 反弹前一天的收盘价
- **最低点定义**: 反弹发生前的最低价（在最高点之后，在起涨点之前）
- **计算**: 距离比 = (起涨点 - 最低点) / 最低点
- **判断**: 距离比 ≤ 15%

#### 3.4 回调支撑条件
- **要求**: 放量长阳后回调不低于长阳线开盘价
- **检查范围**: 从放量长阳日到今天（最新交易日）的所有交易日
- **判断逻辑**: 
  - 获取放量长阳线的开盘价作为支撑位
  - 检查长阳线之后到今天所有交易日的最低价
  - 如果任何一天最低价低于开盘价，则不满足条件
  - 如果所有天最低价不低于开盘价，则满足条件
- **实际含义**: 
  - 确保放量长阳线具有持续的支撑作用
  - 回调不破开盘价说明多头力量强劲，突破有效
  - 避免选择反弹后快速下跌的股票

**实际含义**:
- 股票在最近 10 天内出现了明显的反弹
- 反弹伴随着成交量的放大
- 反弹的起点距离最低点不远（在 15% 以内）
- 这表明反弹是从底部开始的，具有真实性
- 放量长阳线具有支撑作用，回调不破开盘价

**参数**:
- `price_increase_threshold`: 0.08 (8%)
- `volume_ratio_threshold`: 2.5
- `volume_ma_period`: 10 (交易日)

## 过滤条件

### 过滤1：退市股票
- 过滤包含"退"、"未知"、"退市"、"已退"等关键词的股票

### 过滤2：ST 股票
- 过滤以"ST"或"*ST"开头的股票

### 过滤3：数据有效性
- 股票必须有至少 120 个交易日的历史数据
- 最新一天必须有有效的成交量和收盘价

## 参数配置

### 参数分组

#### 深度下跌条件
- `lookback_days`: 回溯天数
  - 默认值：120
  - 范围：60-250
  - 说明：用于寻找最高点和最低点的时间窗口（120个交易日约半年）

- `decline_threshold`: 下跌幅度阈值
  - 默认值：0.45
  - 范围：0.3-0.6
  - 说明：从最高点到最低点的下跌幅度必须超过此值（0.45表示45%）

#### MACD底背离条件
- `macd_divergence_days`: MACD底背离时间窗口
  - 默认值：20
  - 范围：10-40
  - 说明：用于判断底背离的时间窗口（20个交易日）

#### 放量反弹条件
- `price_increase_threshold`: 涨幅阈值
  - 默认值：0.08
  - 范围：0.05-0.15
  - 说明：当日涨幅需要超过此值（0.08表示8%）

- `volume_ratio_threshold`: 成交量倍数阈值
  - 默认值：2.5
  - 范围：1.5-5.0
  - 说明：当日成交量需要是成交量均值的此倍数以上

- `volume_ma_period`: 成交量均值周期
  - 默认值：10
  - 范围：5-20
  - 说明：成交量均值计算周期

1. **启动系统**
   ```bash
   python web_server.py
   ```

2. **访问前端**

   - 在策略选择对话框中勾选"底部趋势拐点"
   - 选择选股逻辑（OR 或 AND）
   - 点击"确认"执行选股

4. **查看结果**
   - 在"选股结果"页面查看选出的股票
   - 点击股票可查看详细信息

### 参数调整

#### 激进策略（选股数量多）
- 降低 `decline_threshold`（如0.3，降低下跌幅度要求）
- 降低 `price_increase_threshold`（如0.05，降低涨幅要求）
- 降低 `volume_ratio_threshold`（如1.5，降低成交量要求）
- 增加 `lookback_days`（如180，延长回溯天数）
- 增加 `macd_divergence_days`（如30，延长底背离判断窗口）

#### 保守策略（选股数量少）
- 提高 `decline_threshold`（如0.6，提高下跌幅度要求）
- 提高 `price_increase_threshold`（如0.12，提高涨幅要求）
- 提高 `volume_ratio_threshold`（如3.0，提高成交量要求）
- 降低 `lookback_days`（如90，缩短回溯天数）
- 降低 `macd_divergence_days`（如15，缩短底背离判断窗口）

## 参数调整指南

### 调整思路

1. **选股数量过多**
   - 提高 `decline_threshold`（下跌幅度要求更高）
   - 提高 `price_increase_threshold`（涨幅要求更高）
   - 提高 `volume_ratio_threshold`（成交量要求更高）
   - 降低 `lookback_days`（缩短回溯天数）
   - 降低 `macd_divergence_days`（缩短底背离判断窗口）

2. **选股数量过少**
   - 降低 `decline_threshold`（下跌幅度要求降低）
   - 降低 `price_increase_threshold`（涨幅要求降低）
   - 降低 `volume_ratio_threshold`（成交量要求降低）
   - 增加 `lookback_days`（延长回溯天数）
   - 增加 `macd_divergence_days`（延长底背离判断窗口）

3. **选股质量不佳（反转不明确）**
   - 提高 `decline_threshold`（确保深度下跌）
   - 降低 `macd_divergence_days`（确保底背离更明显）
   - 提高 `volume_ratio_threshold`（确保成交量配合）
   - 提高 `price_increase_threshold`（确保反弹力度）

### 参数优化建议

- 建议先使用默认参数运行，观察选股结果
- 根据实际选股数量和质量，逐步调整参数
- 每次调整一个参数，观察效果
- 避免过度优化，防止过拟合
- 优先满足深度下跌和MACD底背离条件，放量反弹条件可以适当放宽

## 常见问题

### Q1: 为什么没有选出任何股票？

**可能原因**：
- 参数设置过于严格
- 市场中没有符合条件的股票
- 数据不完整

**解决方案**：
- 降低 `decline_threshold`（如从0.45降至0.3）
- 降低 `price_increase_threshold`（如从0.08降至0.05）
- 降低 `volume_ratio_threshold`（如从2.5降至1.5）
- 增加 `lookback_days`（如从120增至180）
- 检查数据是否完整

### Q2: 选出的股票太多，如何筛选？

**解决方案**：
- 提高 `decline_threshold`（如从0.45提高至0.6）
- 提高 `price_increase_threshold`（如从0.08提高至0.12）
- 提高 `volume_ratio_threshold`（如从2.5提高至3.0）
- 降低 `lookback_days`（如从120降至90）
- 结合其他指标进行二次筛选

### Q3: 策略的准确率如何？

**说明**：
- 策略基于深度下跌和底背离分析，准确率取决于市场环境
- 底部反转信号具有较高的参考价值，但需要结合其他因素
- 建议结合基本面分析进行综合判断
- 建议设置止损点，控制风险
- 成交量配合的反弹通常更可靠

### Q4: 如何理解MACD底背离？

**说明**：
- **底背离定义**：价格创新低，但MACD指标没有创新低
- **实际含义**：价格继续下跌，但下跌动能（MACD）开始减弱
- **技术意义**：表明下跌动能已经衰竭，反转可能即将发生
- **本策略判断**：当前价格 < 前20天平均价格 AND 当前MACD > 前20天平均MACD

### Q5: 三个条件的优先级如何？

**说明**：
- **条件1（深度下跌）**：基础条件，确保股票处于深度下跌状态
- **条件2（MACD底背离）**：确认条件，确保下跌动能衰竭
- **条件3（放量反弹）**：触发条件，确认反转已经开始
- **建议**：优先满足条件1和条件2，条件3可以适当放宽

## 技术细节

### 数据要求
- 最少120个交易日的日线数据
- 包含：开盘价、最高价、最低价、收盘价、成交量

### 计算流程

1. **数据准备**
   - 加载股票日线数据（开盘价、最高价、最低价、收盘价、成交量）
   - 验证数据完整性和有效性

2. **指标计算**
   - 计算MACD指标（DIF、DEA、MACD）
   - 计算KDJ指标（K、D、J值）
   - 计算趋势线（短期趋势、牛熊线）
   - 计算成交量均值（10日均量）

3. **条件检查**
   - **条件1：深度下跌判断**
     - 在120天内找到最高点和最低点
     - 确保高点在前、最低点在后
     - 计算下跌幅度 = (最高价 - 最低价) / 最高价
     - 确认下跌幅度 > 45%
   - **条件2：MACD底背离判断**
     - 计算当前价格和MACD
     - 计算前20天平均价格和平均MACD
     - 确认：当前价格 < 前20天平均价格 AND 当前MACD > 前20天平均MACD
   - **条件3：放量反弹识别**
     - 在最近10个交易日内寻找涨幅>8%的交易日
     - 确认成交量>=2.5倍10日均量
     - 确认起涨点距离最低点<=15%

4. **结果过滤**
   - 过滤 ST、*ST、退市股票
   - 生成选股信号和理由

5. **返回结果**
   - 返回符合条件的股票列表

### 性能指标

- 处理500只股票耗时 < 30秒
- 内存占用 < 500MB
- 支持实时更新

## 选股流程

```
输入：股票日线数据（最新 120+ 个交易日）
  ↓
第1步：数据有效性检查
  - 检查数据长度 ≥ 120 天
  - 检查最新一天有有效成交量和收盘价
  ↓
第2步：过滤检查
  - 过滤退市股票
  - 过滤 ST 股票
  ↓
第3步：条件1检查 - 深度下跌
  - 计算 120 天内最高价和最低价
  - 计算下跌幅度
  - 判断是否 > 45%
  ↓
第4步：条件2检查 - MACD 底背离
  - 计算 MACD 指标
  - 比较当前价格和 MACD 与前 20 天平均值
  - 判断是否满足底背离条件
  ↓
第5步：条件3检查 - 放量反弹
  - 遍历最近 10 个交易日
  - 检查涨幅条件
  - 检查成交量条件
  - 检查距离条件
  - 判断是否同时满足三个条件
  ↓
第6步：综合判断
  - 三个条件都满足 → 生成选股信号
  - 任何条件不满足 → 不选股
  ↓
输出：选股信号（包含价格、成交量比、J值等信息）
```

## 输出信息

当股票满足所有条件时，输出以下信息：

| 字段 | 说明 |
|------|------|
| `date` | 选股日期 |
| `close` | 收盘价 |
| `J` | KDJ 指标中的 J 值 |
| `volume_ratio` | 成交量比（当日成交量 / 10 日均量） |
| `market_cap` | 市值（亿元） |
| `short_term_trend` | 短期趋势线 |
| `bull_bear_line` | 多空线 |
| `reasons` | 选股原因 |

## 策略特点

### 优点
1. **多指标组合**: 使用深度下跌、MACD 底背离、放量反弹三个指标
2. **严格的选股条件**: 三个条件都必须满足，确保选股质量
3. **底部反转逻辑**: 从多个角度确认底部反转机会
4. **实时性强**: 基于最新的日线数据进行判断

### 缺点
1. **选股能力受限**: 当前市场环境下，满足所有条件的股票较少
2. **市场环境依赖**: 在下跌通道中，放量反弹条件难以满足
3. **参数敏感**: 参数调整会显著影响选股结果
