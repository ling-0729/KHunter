## 推荐新策略（按优先级排序）

### 优先级1：均线金叉策略 ⭐⭐⭐⭐⭐

**推荐理由**：
- 均线系统是最经典的技术分析工具
- 与现有策略互补性强
- 实现简单，参数少
- 适用范围广
- 成功率较高

**策略描述**：
识别短期均线上穿长期均线的金叉信号，捕捉趋势反转机会。

**选股条件**：
1. **金叉确认**：短期均线（如5日）上穿长期均线（如20日）
2. **均线方向**：短期均线和长期均线都向上
3. **成交量配合**：金叉当日成交量放大
4. **价格位置**：收盘价在所有均线上方
5. **乖离控制**：价格偏离均线不超过一定比例

**参数配置**：
- `short_ma_period`: 短期均线周期（默认5）
- `long_ma_period`: 长期均线周期（默认20）
- `volume_ratio`: 成交量倍数（默认1.5）
- `deviation_threshold`: 乖离阈值（默认5%）

**适用场景**：
- 趋势反转初期
- 上升趋势确认
- 中长期投资

**风险等级**：低

**实现难度**：⭐（简单）

### 优先级2：突破阻力位策略 ⭐⭐⭐⭐⭐

**推荐理由**：
- 突破是最强烈的买入信号
- 与现有策略互补
- 实现难度适中
- 成功率高
- 适合短线操作

**策略描述**：
识别股价突破关键阻力位（如前高、箱体上沿）的信号，捕捉上涨突破机会。

**选股条件**：
1. **突破确认**：收盘价突破前N日最高价
2. **成交量放大**：突破日成交量显著放大
3. **站稳确认**：突破后连续3天收盘价在阻力位上方
4. **回踩不破**：突破后如有回调，不跌破阻力位
5. **趋势配合**：短期趋势向上

**参数配置**：
- `lookback_days`: 回溯天数（默认20）
- `breakout_ratio`: 突破幅度（默认1%）
- `volume_ratio`: 成交量倍数（默认2.0）
- `hold_days`: 站稳天数（默认3）
- `pullback_ratio`: 回踩比例（默认2%）

**适用场景**：
- 横盘整理后突破
- 上升趋势加速
- 短线操作

**风险等级**：中等

**实现难度**：⭐⭐（中等）

### 优先级3：缩量回调策略 ⭐⭐⭐⭐

**推荐理由**：
- 上升趋势中的回调是最佳买入时机
- 与碗口反弹策略形成互补
- 实现难度适中
- 风险可控
- 成功率较高

**策略描述**：
识别上升趋势中的缩量回调机会，在回调企稳时买入。

**选股条件**：
1. **上升趋势**：短期均线 > 长期均线
2. **缩量回调**：回调时成交量显著萎缩
3. **回调幅度**：回调幅度在合理范围内（如10-20%）
4. **企稳信号**：成交量开始放大，价格止跌
5. **支撑确认**：在关键支撑位（如均线）企稳

**参数配置**：
- `short_ma_period`: 短期均线周期（默认10）
- `long_ma_period`: 长期均线周期（默认30）
- `volume_shrink_ratio`: 缩量比例（默认0.5）
- `pullback_ratio`: 回调幅度（默认15%）
- `support_days`: 企稳天数（默认2）

**适用场景**：
- 上升趋势中的回调
- 中长期投资
- 稳健操作

**风险等级**：低

**实现难度**：⭐⭐（中等）

## 详细设计：均线金叉策略

### 策略概述

均线金叉策略是最经典的技术分析策略之一，通过识别短期均线上穿长期均线的金叉信号，捕捉趋势反转机会。

### 选股条件

#### 条件1：金叉确认
- 短期均线（如5日）上穿长期均线（如20日）
- 金叉发生在最近N天内（默认5天）

#### 条件2：均线方向
- 短期均线向上
- 长期均线向上或走平

#### 条件3：成交量配合
- 金叉当日成交量 >= 前N日平均成交量 × volume_ratio
- 默认：成交量 >= 前5日均量 × 1.5

#### 条件4：价格位置
- 收盘价在所有均线上方
- 确保价格已经站稳

#### 条件5：乖离控制
- 价格偏离短期均线不超过 deviation_threshold
- 默认：偏离不超过5%

### 参数配置

| 参数名 | 默认值 | 范围 | 说明 |
|--------|--------|------|------|
| `short_ma_period` | 5 | 3-10 | 短期均线周期 |
| `long_ma_period` | 20 | 10-60 | 长期均线周期 |
| `gold_cross_days` | 5 | 1-10 | 金叉发生天数 |
| `volume_ratio` | 1.5 | 1.0-3.0 | 成交量倍数 |
| `volume_ma_period` | 5 | 3-10 | 成交量均值周期 |
| `deviation_threshold` | 0.05 | 0.02-0.15 | 乖离阈值（5%） |

### 实现要点

1. **均线计算**：使用MA函数计算短期和长期均线
2. **金叉识别**：判断短期均线是否上穿长期均线
3. **方向判断**：计算均线的斜率，判断方向
4. **成交量判断**：计算成交量均值，判断是否放量
5. **乖离计算**：计算价格与均线的偏离度

### 伪代码

```python
def select_stocks(df, params):
    # 计算均线
    short_ma = MA(df['close'], params['short_ma_period'])
    long_ma = MA(df['close'], params['long_ma_period'])
    
    # 判断金叉
    gold_cross = (short_ma > long_ma) & (REF(short_ma, 1) <= REF(long_ma, 1))
    
    # 判断均线方向
    short_ma_up = short_ma > REF(short_ma, 1)
    long_ma_up = long_ma >= REF(long_ma, 1)
    
    # 判断成交量
    volume_ma = MA(df['volume'], params['volume_ma_period'])
    volume_surge = df['volume'] >= volume_ma * params['volume_ratio']
    
    # 判断乖离
    deviation = abs(df['close'] - short_ma) / short_ma
    deviation_ok = deviation <= params['deviation_threshold']
    
    # 综合判断
    signal = gold_cross & short_ma_up & long_ma_up & volume_surge & deviation_ok
    
    return signal
```

- **选股质量**：高
- **风险等级**：低
- **适用场景**：趋势反转、上升趋势确认

## 详细设计：突破阻力位策略

### 策略概述

突破阻力位策略通过识别股价突破关键阻力位的信号，捕捉上涨突破机会。突破是最强烈的买入信号之一。

### 选股条件

#### 条件1：突破确认
- 收盘价突破前N日最高价 × (1 + breakout_ratio)
- 默认：突破前20日最高价 × 1.01（1%）

#### 条件2：成交量放大
- 突破日成交量 >= 前N日平均成交量 × volume_ratio
- 默认：成交量 >= 前10日均量 × 2.0

#### 条件3：站稳确认
- 突破后连续hold_days天收盘价在阻力位上方
- 默认：连续3天站稳

#### 条件4：回踩不破
- 突破后如有回调，不跌破阻力位 × (1 - pullback_ratio)
- 默认：回踩不超过2%

#### 条件5：趋势配合
- 短期趋势向上
- 确保突破不是假突破

### 参数配置

| 参数名 | 默认值 | 范围 | 说明 |
|--------|--------|------|------|
| `lookback_days` | 20 | 10-60 | 回溯天数 |
| `breakout_ratio` | 0.01 | 0.005-0.03 | 突破幅度（1%） |
| `volume_ratio` | 2.0 | 1.5-5.0 | 成交量倍数 |
| `volume_ma_period` | 10 | 5-20 | 成交量均值周期 |
| `hold_days` | 3 | 2-5 | 站稳天数 |
| `pullback_ratio` | 0.02 | 0.01-0.05 | 回踩比例（2%） |

### 实现要点

1. **阻力位计算**：计算前N日最高价作为阻力位
2. **突破识别**：判断收盘价是否突破阻力位
3. **成交量判断**：计算成交量均值，判断是否放量
4. **站稳确认**：检查突破后是否连续站稳
5. **回踩检查**：检查是否有回踩且不破阻力位

### 预期效果

- **选股数量**：中等偏少（5-20只）
- **选股质量**：高
- **风险等级**：中等
- **适用场景**：横盘整理后突破、上升趋势加速

1. **优先开发均线金叉策略**：
   - 实现简单，参数少
   - 成功率高，适用范围广
   - 与现有策略形成良好互补

   - 适合短线操作
   - 可以提高选股多样性

3. **根据实际需求选择后续策略**：

   - 如果追求激进：开发强势股策略

### 注意事项

1. **逐步开发**：不要一次性开发太多策略
2. **充分测试**：每个策略都要进行充分测试
