## 新增推荐策略（按优先级排序）

### 优先级1：资金流向策略 ⭐⭐⭐⭐⭐

**推荐理由**：
- 资金是股价上涨的核心驱动力
- 与现有纯技术分析策略形成互补
- 实现难度适中
- 成功率较高
- 适合捕捉主力资金动向

**策略描述**：
识别主力资金持续流入的股票，捕捉资金推动的上涨机会。

**选股条件**：
1. **资金净流入**：连续N天资金净流入
2. **大单比例**：大单成交占比高于阈值
3. **成交量配合**：成交量温和放大
4. **趋势确认**：价格趋势向上
5. **风险控制**：避免高位追涨

**参数配置**：
- `net_flow_days`: 资金净流入天数（默认3）
- `large_order_ratio`: 大单比例（默认0.6）
- `volume_ratio`: 成交量倍数（默认1.2）
- `price_change_threshold`: 价格涨幅阈值（默认5%）

**适用场景**：
- 主力建仓期
- 拉升初期
- 中短线操作

**风险等级**：中等

**实现难度**：⭐⭐（中等）

### 优先级2：板块轮动策略 ⭐⭐⭐⭐⭐

**推荐理由**：
- 板块轮动是A股市场的重要特征
- 与个股策略形成互补
- 实现难度适中
- 适合把握市场热点
- 成功率较高

**策略描述**：
识别板块轮动机会，选择领涨板块中的龙头股。

**选股条件**：
1. **板块强势**：板块指数近期涨幅领先
2. **个股龙头**：个股在板块中涨幅居前
3. **成交量配合**：板块和个股成交量放大
4. **趋势确认**：板块和个股趋势向上
5. **联动性**：板块内个股联动性强

**参数配置**：
- `sector_outperform_days`: 板块领先天数（默认5）
- `stock_leading_ratio`: 个股领先比例（默认0.3）
- `volume_ratio`: 成交量倍数（默认1.5）
- `sector_correlation`: 板块联动性（默认0.7）

**适用场景**：
- 市场热点切换期
- 行业政策利好期
- 中短线操作

**风险等级**：中等

**实现难度**：⭐⭐（中等）

### 优先级3：事件驱动策略 ⭐⭐⭐⭐

**推荐理由**：
- 事件是股价波动的重要催化剂
- 与技术分析策略形成互补
- 实现难度适中
- 适合捕捉短期机会
- 成功率中等

**策略描述**：
识别重大事件（如业绩预告、资产重组、政策利好等）驱动的股票机会。

**选股条件**：
1. **事件确认**：发生重大利好事件
2. **资金反应**：事件后资金快速流入
3. **价格表现**：事件后价格上涨
4. **成交量配合**：事件后成交量放大
5. **持续性**：事件影响具有持续性

**参数配置**：
- `event_type`: 事件类型（业绩、重组、政策等）
- `price_change_threshold`: 价格涨幅阈值（默认5%）
- `volume_ratio`: 成交量倍数（默认2.0）
- `event_days`: 事件发生天数（默认3）

**适用场景**：
- 业绩发布期
- 政策出台期
- 资产重组期
- 短线操作

**风险等级**：中高

**实现难度**：⭐⭐（中等）

### 优先级4：波动率策略 ⭐⭐⭐⭐

**推荐理由**：
- 波动率是风险和机会的重要指标
- 与趋势策略形成互补
- 实现难度适中
- 适合震荡市场
- 成功率中等

**策略描述**：
识别波动率突破的股票，捕捉高波动带来的交易机会。

**选股条件**：
1. **波动率突破**：波动率创近期新高
2. **价格突破**：价格突破重要阻力位
3. **成交量配合**：成交量放大
4. **趋势确认**：短期趋势向上
5. **风险控制**：设置合理止损

**参数配置**：
- `volatility_period`: 波动率计算周期（默认20）
- `volatility_ratio`: 波动率倍数（默认1.5）
- `volume_ratio`: 成交量倍数（默认1.5）
- `price_breakout_ratio`: 价格突破幅度（默认1%）

**适用场景**：
- 震荡市场
- 重要事件前后
- 中短线操作

**风险等级**：中高

**实现难度**：⭐⭐（中等）

### 优先级5：多因子策略 ⭐⭐⭐

**推荐理由**：
- 多因子策略综合多个维度的信息
- 与单一因子策略形成互补
- 实现难度较高
- 适合长期投资
- 成功率较高

**策略描述**：
综合技术面、基本面、资金面等多个因子，构建综合评分体系选股。

**选股条件**：
1. **技术因子**：均线金叉、突破等技术信号
2. **基本面因子**：业绩增长、估值合理
3. **资金因子**：资金净流入、主力持仓
4. **情绪因子**：市场情绪、换手率
5. **综合评分**：多因子综合评分高于阈值

**参数配置**：
- `factor_weights`: 各因子权重
- `score_threshold`: 综合评分阈值（默认0.7）
- `holding_period`: 持有周期（默认30）
- `rebalance_frequency`: 再平衡频率（默认7）

**适用场景**：
- 中长期投资
- 稳健型投资
- 价值投资

**风险等级**：低

**实现难度**：⭐⭐⭐（较难）

**推荐理由**：
- 季节性规律是市场的重要特征
- 与其他策略形成互补
- 实现难度适中
- 适合特定时期操作
- 成功率中等

**策略描述**：
识别具有明显季节性规律的股票，在特定时期布局。

**选股条件**：
1. **季节性确认**：历史上该时期表现优异
2. **趋势配合**：当前趋势向上
3. **成交量配合**：成交量开始放大
4. **基本面支撑**：基本面良好
5. **风险控制**：设置止损点

**参数配置**：
- `seasonal_period`: 季节性周期（默认30）
- `historical_outperformance`: 历史表现（默认0.1）
- `volume_ratio`: 成交量倍数（默认1.2）
- `price_change_threshold`: 价格涨幅阈值（默认3%）

**适用场景**：
- 特定季节
- 行业旺季
- 事件驱动期

**风险等级**：中等

**实现难度**：⭐⭐（中等）

## 开发建议

1. **优先开发资金流向策略**：
   - 资金是股价上涨的核心驱动力
   - 与现有策略形成良好互补
   - 实现相对简单

2. **其次开发板块轮动策略**：
   - 板块轮动是A股市场的重要特征
   - 适合把握市场热点
   - 可以提高选股的时效性

3. **根据实际需求选择后续策略**：
   - 如果追求短期机会：开发事件驱动策略
   - 如果追求稳健：开发多因子策略
   - 如果追求特定时期机会：开发季节性策略

4. **注意事项**：
   - 每个策略都要进行充分测试
   - 根据市场环境调整参数
   - 集成风险控制机制
   - 持续优化策略性能
