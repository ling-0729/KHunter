# 碗口反弹策略说明书

## 策略概述

碗口反弹策略是一种基于趋势线和成交量分析的量化选股策略，通过识别价格在上升趋势中的回调买入机会，捕捉"碗口"形态的反弹信号。

### 策略特点

- **趋势跟踪**：基于知行趋势线判断上升趋势
- **位置精准**：识别价格在趋势线之间的相对位置
- **量价配合**：结合成交量异动确认信号
- **分类标记**：按优先级分类（回落碗中 > 靠近多空线 > 靠近短期趋势线）

## 选股条件

策略通过以下四个核心条件的组合来识别碗口反弹信号：

### 条件1：上升趋势判断

使用知行趋势线系统判断股票是否处于上升趋势。

**判断方法**：
- 计算知行短期趋势线：EMA(EMA(CLOSE,10),10)
- 计算知行多空线：(MA(CLOSE,5) + MA(CLOSE,10) + MA(CLOSE,20) + MA(CLOSE,30)) / 4
- 上升趋势确认：知行短期趋势线 > 知行多空线

**技术原理**：
- 短期趋势线反映短期价格走势
- 多空线反映中长期趋势
- 短期趋势线在多空线上方表示上升趋势

**参数**：
- `M1`: MA周期1（默认14）
- `M2`: MA周期2（默认28）
- `M3`: MA周期3（默认57）
- `M4`: MA周期4（默认114）

### 条件2：异动放量阳线

识别近期出现的放量阳线，作为异动信号。

**判断方法**：
- 成交量 >= 前一日成交量 × N（N默认为4）
- 阳线：收盘价 > 开盘价
- 总市值 > CAP（CAP默认为40亿元）

**参数**：
- `N`: 成交量倍数（默认4）
- `CAP`: 总市值阈值（默认40亿元）
- `M`: 回溯天数（默认15），在此天数内寻找异动

### 条件3：J值低位

使用KDJ指标判断超卖状态。

**判断方法**：
- 计算KDJ指标（9,3,3参数）
- J值 <= J_VAL（J_VAL默认为30）

**技术原理**：
- J值是KDJ指标中最敏感的指标
- J值低位表示超卖，存在反弹机会

**参数**：
- `J_VAL`: J值上限（默认30）

### 条件4：位置分类标记

根据价格相对于趋势线的位置进行分类标记，按优先级判断。

**分类标准**（按优先级从高到低）：

1. **回落碗中**（优先级最高）
   - 价格位于知行多空线和知行短期趋势线之间
   - 表示价格从短期趋势线回调到多空线附近
   - 形成类似"碗口"的形态

2. **靠近多空线**
   - 价格距离知行多空线 ±duokong_pct% 范围内
   - duokong_pct默认为3%
   - 表示价格接近中长期支撑位

3. **靠近短期趋势线**
   - 价格距离知行短期趋势线 ±short_pct% 范围内
   - short_pct默认为2%
   - 表示价格接近短期压力位

**参数**：
- `duokong_pct`: 距离多空线百分比（默认3%）
- `short_pct`: 距离短期趋势线百分比（默认2%）

## 参数配置

### 参数分组

#### 趋势条件
- `M1`: MA周期1
  - 默认值：14
  - 范围：5-30
  - 说明：多空线计算的第一条均线周期

- `M2`: MA周期2
  - 默认值：28
  - 范围：10-60
  - 说明：多空线计算的第二条均线周期

- `M3`: MA周期3
  - 默认值：57
  - 范围：20-120
  - 说明：多空线计算的第三条均线周期

- `M4`: MA周期4
  - 默认值：114
  - 范围：50-250
  - 说明：多空线计算的第四条均线周期

#### 异动条件
- `N`: 成交量倍数
  - 默认值：4
  - 范围：1.5-10
  - 说明：成交量需要是前一日成交量的此倍数以上

- `M`: 回溯天数
  - 默认值：15
  - 范围：1-30
  - 说明：在此天数内寻找异动放量阳线

- `CAP`: 总市值阈值
  - 默认值：40（单位：亿元）
  - 范围：10-500
  - 说明：股票总市值必须大于此值（单位：亿元）

#### KDJ条件
- `J_VAL`: J值上限
  - 默认值：30
  - 范围：10-50
  - 说明：J值必须小于等于此值才认为超卖

#### 位置条件
- `duokong_pct`: 距离多空线百分比
  - 默认值：3
  - 范围：1-10
  - 说明：价格距离多空线的百分比范围（±）

- `short_pct`: 距离短期趋势线百分比
  - 默认值：2
  - 范围：1-10
  - 说明：价格距离短期趋势线的百分比范围（±）

## 使用指南

### 基本使用

1. **启动系统**
   ```bash
   python web_server.py
   ```

2. **访问前端**
   - 打开浏览器，访问 http://localhost:5000

3. **执行选股**
   - 点击"执行选股"按钮
   - 在策略选择对话框中勾选"碗口反弹策略"
   - 选择选股逻辑（OR 或 AND）
   - 点击"确认"执行选股

4. **查看结果**
   - 在"选股结果"页面查看选出的股票
   - 点击股票可查看详细信息
   - 查看分类标记（回落碗中、靠近多空线、靠近短期趋势线）

### 参数调整

#### 激进策略（选股数量多）
- 降低 `N`（如2，降低成交量要求）
- 增加 `M`（如30，延长异动回溯天数）
- 降低 `CAP`（如20，降低市值要求）
- 提高 `J_VAL`（如50，放宽J值要求）
- 增加 `duokong_pct`（如5%，扩大多空线范围）
- 增加 `short_pct`（如3%，扩大短期趋势线范围）

#### 保守策略（选股数量少）
- 提高 `N`（如6，提高成交量要求）
- 降低 `M`（如5，缩短异动回溯天数）
- 提高 `CAP`（如100，提高市值要求）
- 降低 `J_VAL`（如20，提高J值要求）
- 降低 `duokong_pct`（如1%，缩小多空线范围）
- 降低 `short_pct`（如1%，缩小短期趋势线范围）

## 参数调整指南

### 调整思路

1. **选股数量过多**
   - 提高 `N`（成交量要求更高）
   - 降低 `M`（缩短异动回溯天数）
   - 提高 `CAP`（提高市值要求）
   - 降低 `J_VAL`（提高J值要求）
   - 降低 `duokong_pct` 和 `short_pct`（缩小位置范围）

2. **选股数量过少**
   - 降低 `N`（成交量要求降低）
   - 提高 `M`（延长异动回溯天数）
   - 降低 `CAP`（降低市值要求）
   - 提高 `J_VAL`（放宽J值要求）
   - 提高 `duokong_pct` 和 `short_pct`（扩大位置范围）

3. **选股质量不佳（趋势不明确）**
   - 调整 `M1`、`M2`、`M3`、`M4`（优化趋势线参数）
   - 提高 `N`（确保成交量异动明显）
   - 降低 `J_VAL`（确保超卖状态）

### 参数优化建议

- 建议先使用默认参数运行，观察选股结果
- 根据实际选股数量和质量，逐步调整参数
- 每次调整一个参数，观察效果
- 避免过度优化，防止过拟合
- 关注分类标记的分布，优先选择"回落碗中"的股票

## 常见问题

### Q1: 为什么没有选出任何股票？

**可能原因**：
- 参数设置过于严格
- 市场中没有符合条件的股票
- 数据不完整

**解决方案**：
- 降低 `N` 和 `CAP`
- 提高 `J_VAL`
- 增加 `M`
- 检查数据是否完整

### Q2: 选出的股票太多，如何筛选？

**解决方案**：
- 提高 `N` 和 `CAP`
- 降低 `M`
- 降低 `J_VAL`
- 优先选择"回落碗中"分类的股票
- 结合其他指标进行二次筛选

### Q3: 策略的准确率如何？

**说明**：
- 策略基于趋势线和成交量分析，准确率取决于市场环境
- 建议结合基本面分析进行综合判断
- 建议设置止损点，控制风险
- "回落碗中"分类的股票通常质量更高

### Q4: 如何理解分类标记？

**说明**：
- **回落碗中**：价格从短期趋势线回调到多空线，形成"碗口"形态，反弹概率最高
- **靠近多空线**：价格接近中长期支撑位，存在支撑
- **靠近短期趋势线**：价格接近短期压力位，可能突破

## 技术细节

### 数据要求
- 最少30个交易日的日线数据
- 包含：开盘价、最高价、最低价、收盘价、成交量

### 计算流程

1. **数据准备**
   - 加载股票日线数据
   - 验证数据完整性和有效性

2. **指标计算**
   - 计算知行短期趋势线（EMA(EMA(CLOSE,10),10)）
   - 计算知行多空线（MA5+MA10+MA20+MA30）/4
   - 计算KDJ指标（K、D、J值）
   - 计算成交量比例

3. **条件检查**
   - **条件1：上升趋势判断**
     - 检查知行短期趋势线 > 知行多空线
   - **条件2：异动放量阳线识别**
     - 在M天内寻找成交量>=前一日×N的阳线
     - 确认总市值>CAP
   - **条件3：J值低位检查**
     - 确认J值<=J_VAL
   - **条件4：位置分类标记**
     - 判断价格相对趋势线的位置
     - 按优先级分类（回落碗中 > 靠近多空线 > 靠近短期趋势线）

4. **结果过滤**
   - 过滤 ST、*ST、退市股票
   - 过滤数据异常的股票
   - 生成选股信号和分类标记

5. **返回结果**
   - 返回符合条件的股票列表
   - 包含分类标记信息
