# 形态猎手 KHunter - Logo 设计说明

## 概述

为"形态猎手KHunter"系统设计了专业的Logo和Favicon，体现系统的核心特性和品牌形象。

## Logo 文件

### 1. 主Logo
- **文件路径**: `web/static/images/logo.svg`
- **尺寸**: 200x200px (可缩放)
- **格式**: SVG (矢量格式，支持任意缩放)
- **用途**: 系统侧边栏、文档、营销材料等

### 2. Favicon (网站图标)
- **文件路径**: `web/static/images/favicon.svg`
- **尺寸**: 64x64px (可缩放)
- **格式**: SVG (矢量格式)
- **用途**: 浏览器标签页图标、书签图标

### 3. Logo 预览页面
- **文件路径**: `web/static/images/logo-preview.html`
- **用途**: 查看Logo设计效果和设计说明

## 设计元素

### 核心形象：猎手 + 股票分析

```
┌─────────────────────────────────────┐
│                                     │
│         △ (猎手的帽子)              │
│        ╱ ╲                          │
│       ╱   ╲                         │
│      ╱     ╲                        │
│     ╱       ╲                       │
│    ╱─────────╲                      │
│   │  K线图表  │  (股票分析)         │
│   │ ▁ ▂ ▃ ▄  │                      │
│    ╲         ╱                      │
│     ╲       ╱                       │
│      ╲     ╱                        │
│       ╲   ╱                         │
│        ╲ ╱                          │
│         ▼ (猎手的腿)                │
│                                     │
│      ↗ (上升箭头 - 收益)            │
│                                     │
└─────────────────────────────────────┘
```

### 设计含义

| 元素 | 含义 | 象征 |
|------|------|------|
| 三角形头部 | 猎手的帽子 | 专业、敏锐、目标明确 |
| 身体 | K线图 | 股票价格变动 |
| 四条K线 | 股票分析 | 系统的核心功能 |
| 上升箭头 | 收益方向 | 帮助用户获利 |
| 外圆环 | 完整性 | 系统的完整性和稳定性 |
| 虚线趋势 | 上升趋势 | 正向发展 |

## 色彩方案

### 主色系

| 颜色 | 十六进制 | RGB | 含义 |
|------|---------|-----|------|
| 深蓝 | #1e3a8a | 30, 58, 138 | 专业、稳定、信任 |
| 蓝色 | #2563eb | 37, 99, 235 | 主体颜色、猎手形象 |
| 绿色 | #10b981 | 16, 185, 129 | 上升、收益、正向 |
| 红色 | #ef4444 | 239, 68, 68 | 下跌、风险、警示 |
| 橙色 | #f59e0b | 245, 158, 11 | 中立、平衡、过渡 |

### 色彩应用

- **深蓝色**: 背景圆形、外圆环、文字
- **蓝色**: 猎手形象（头部、身体、腿、武器）
- **绿色**: 上升的K线、上升箭头
- **红色**: 下跌的K线
- **橙色**: 平衡的K线

## HTML 集成

### 在网页中使用

```html
<!-- Favicon (浏览器标签页) -->
<link rel="icon" type="image/svg+xml" href="/static/images/favicon.svg">

<!-- Logo 图片 -->
<img src="/static/images/logo.svg" alt="KHunter Logo" class="logo-img">
```

### 在侧边栏中的使用

```html
<div class="logo">
    <img src="/static/images/logo.svg" alt="KHunter Logo" class="logo-img" 
         style="width: 60px; height: 60px; margin-bottom: 10px;">
    <h1>形态猎手KHunter</h1>
</div>
```

## 设计特点

### 优势

1. **矢量格式**: SVG格式支持任意缩放，不失真
2. **简洁专业**: 设计简洁，易于识别和记忆
3. **含义丰富**: 将"猎手"和"股票分析"完美结合
4. **色彩协调**: 色彩搭配专业，符合金融行业审美
5. **易于扩展**: 可轻松调整尺寸、颜色等属性

### 应用场景

- ✅ 浏览器标签页图标 (Favicon)
- ✅ 系统侧边栏Logo
- ✅ 文档和报告封面
- ✅ 营销材料和宣传
- ✅ 社交媒体头像
- ✅ 应用程序图标
- ✅ 打印物料
