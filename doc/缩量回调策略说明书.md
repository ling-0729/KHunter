# 缩量回调策略说明书

## 策略概述

缩量回调策略是一种基于成交量和价格回调的量化选股策略。该策略通过识别股票在上涨趋势中的缩量回调，寻找可能的买入机会，具有较高的成功率和风险收益比。

### 策略特点

- **趋势识别**：基于均线系统判断股票的整体趋势
- **量价配合**：强调成交量与价格的配合关系
- **回调筛选**：识别健康的缩量回调形态
- **企稳确认**：通过多维度指标确认企稳信号
- **风险可控**：明确的回调幅度限制和止损点设置

## 选股条件

策略通过以下四个关键条件的组合来识别缩量回调形态：

### 条件1：趋势判断

确保股票处于整体上升趋势中，为回调后上涨提供基础。

**判断方法**：
- 短期均线 > 长期均线 * 0.95（允许5%的偏离）
- 收盘价不低于长期均线太多（允许8%的偏离，适应回调）

**参数**：
- `short_ma_period`: 短期均线周期（默认10）
- `long_ma_period`: 长期均线周期（默认30）

### 条件2：回调幅度

识别合理范围内的回调，避免过度回调或回调不足。

**判断方法**：
- 回调幅度 >= 3%（确保有足够的调整）
- 回调幅度 <= 15%（避免过度调整）

**参数**：
- `pullback_min_ratio`: 最小回调幅度（默认0.03 = 3%）
- `pullback_max_ratio`: 最大回调幅度（默认0.15 = 15%）
- `lookback_days`: 回溯天数（默认10）

### 条件3：缩量状态

确认回调过程中成交量萎缩，表明抛压减轻。

**判断方法**：
- 成交量比 < 1.0（相对5日均量缩量）
- 成交量比 > 0.3（避免成交量过低导致的流动性问题）

**参数**：
- `volume_ma_period`: 成交量均线周期（默认5）
- `volume_shrink_ratio`: 缩量比例（默认0.7）

### 条件4：企稳信号

确认价格已经企稳，为上涨提供支撑。

**判断方法**：
- 价格不再创新低（连续support_days天收盘价不低于最低点）
- 收盘价开始回升
- 成交量保持在相对低位或突然放大（可能是反弹信号）

**参数**：
- `support_days`: 企稳天数（默认2）
- `support_ratio`: 支撑比例（默认0.02）

## 参数配置

### 参数分组

#### 均线参数
- `short_ma_period`: 短期均线周期
  - 默认值：10
  - 范围：5-20
  - 说明：短期趋势判断的均线周期

- `long_ma_period`: 长期均线周期
  - 默认值：30
  - 范围：20-60
  - 说明：长期趋势判断的均线周期

#### 成交量参数
- `volume_ma_period`: 成交量均线周期
  - 默认值：5
  - 范围：3-10
  - 说明：计算成交量比的均线周期

- `volume_shrink_ratio`: 缩量比例
  - 默认值：0.7
  - 范围：0.5-0.9
  - 说明：缩量回调的成交量比例要求

- `volume_expand_ratio`: 放量比例
  - 默认值：1.0
  - 范围：0.8-2.0
  - 说明：企稳时的成交量放大要求

#### 回调参数
- `pullback_min_ratio`: 最小回调幅度
  - 默认值：0.03
  - 范围：0.01-0.05
  - 说明：回调幅度的最小值（小数形式）

- `pullback_max_ratio`: 最大回调幅度
  - 默认值：0.15
  - 范围：0.10-0.20
  - 说明：回调幅度的最大值（小数形式）

- `lookback_days`: 回溯天数
  - 默认值：10
  - 范围：5-20
  - 说明：计算回调幅度的回溯天数

#### 企稳参数
- `support_days`: 企稳天数
  - 默认值：2
  - 范围：1-5
  - 说明：确认企稳的连续天数

- `support_ratio`: 支撑比例
  - 默认值：0.02
  - 范围：0.01-0.05
  - 说明：企稳时的价格支撑强度

#### 市值参数
- `min_market_cap`: 最小市值(亿)
  - 默认值：20
  - 范围：10-100
  - 说明：股票总市值的最小值（亿元）

- `max_market_cap`: 最大市值(亿)
  - 默认值：1000
  - 范围：100-5000
  - 说明：股票总市值的最大值（亿元）

## 使用指南

### 基本使用

1. **启动系统**
   ```bash
   python web_server.py
   ```

2. **访问前端**
   - 打开浏览器，访问 http://localhost:5000

3. **执行选股**
   - 点击"执行选股"按钮
   - 在策略选择对话框中勾选"缩量回调策略"
   - 选择选股逻辑（OR 或 AND）
   - 点击"确认"执行选股

4. **查看结果**
   - 在"选股结果"页面查看选出的股票
   - 点击股票可查看详细K线图

### 参数调整

- 增加 `volume_shrink_ratio`（如0.9）

- 降低 `pullback_max_ratio`（如0.10）

- 缩小市值范围（降低 `max_market_cap`）

## 参数调整指南

### 调整思路

1. **选股数量过多**
   - 提高 `pullback_min_ratio`（回调要求更高）
   - 降低 `pullback_max_ratio`（回调幅度限制更严格）
   - 降低 `volume_shrink_ratio`（缩量要求更严格）
   - 增加 `support_days`（企稳要求更严格）
   - 缩小市值范围（降低 `max_market_cap`）

2. **选股数量过少**
   - 降低 `pullback_min_ratio`（回调要求降低）
   - 增加 `pullback_max_ratio`（回调幅度限制放宽）
   - 增加 `volume_shrink_ratio`（缩量要求放松）
   - 减少 `support_days`（企稳要求放松）
   - 放宽市值限制（增加 `max_market_cap`）

3. **选股质量不佳**
   - 增加 `long_ma_period`（更严格的趋势判断）
   - 降低 `pullback_max_ratio`（避免过度回调）
   - 降低 `volume_shrink_ratio`（要求更明显的缩量）
   - 增加 `support_days`（更确认的企稳信号）
   - 结合其他策略进行二次筛选

### 参数优化建议

- 建议先使用默认参数运行，观察选股结果
- 根据实际选股数量和质量，逐步调整参数
- 每次调整一个参数，观察效果
- 避免过度优化，防止过拟合
- 根据市场环境灵活调整参数（牛市可适当放宽，熊市可适当严格）

## 形态分类

根据回调幅度和成交量特征，将缩量回调形态分为三类：

### 强势缩量回调
- **回调幅度**：3%-7%
- **成交量**：明显缩量（成交量比 < 0.7）
- **特征**：回调幅度小，缩量明显，多头力量强
- **风险**：风险较低，上涨概率高
- **建议**：适合稳健投资者

### 标准缩量回调
- **回调幅度**：7%-12%
- **成交量**：适度缩量（成交量比 0.7-0.9）
- **特征**：回调幅度适中，成交量配合良好
- **风险**：风险收益比合理
- **建议**：适合大多数投资者

### 深度缩量回调
- **回调幅度**：12%-15%
- **成交量**：极度缩量（成交量比 < 0.6）
- **特征**：回调幅度较大，成交量极度萎缩
- **风险**：风险较高，但可能有更大的反弹空间
- **建议**：适合激进投资者，需设置严格止损

## 常见问题

### Q1: 为什么没有选出任何股票？

**可能原因**：
- 参数设置过于严格
- 市场处于下跌趋势，缺乏符合条件的股票
- 数据不完整

**解决方案**：
- 降低 `pullback_min_ratio`（如0.01）
- 增加 `pullback_max_ratio`（如0.20）
- 增加 `volume_shrink_ratio`（如0.9）
- 减少 `support_days`（如1）
- 检查数据是否完整

### Q2: 选出的股票太多，如何筛选？

**解决方案**：
- 提高 `pullback_min_ratio`（如0.05）
- 降低 `pullback_max_ratio`（如0.10）
- 降低 `volume_shrink_ratio`（如0.6）
- 增加 `support_days`（如3）
- 缩小市值范围（降低 `max_market_cap`）
- 结合其他指标进行二次筛选

### Q3: 策略的准确率如何？

**说明**：
- 缩量回调是经典的量价配合形态，市场认可度高
- 准确率取决于市场环境和个股情况
- 建议结合基本面分析进行综合判断
- 建议设置止损点，控制风险

### Q4: 如何设置止损点？

**建议**：
- 在回调低点设置止损（通常在回调幅度的1.5倍处）
- 止损幅度通常在5%-8%
- 严格执行止损，避免深度套牢
- 可结合技术指标（如MACD、KDJ）调整止损点

## 技术细节

### 数据要求

- 最少30个交易日的日线数据
- 包含：开盘价、最高价、最低价、收盘价、成交量

### 计算流程

1. 计算指标（均线、成交量比、回调幅度等）
2. 检查趋势判断条件（短期均线 > 长期均线 * 0.95）
3. 计算回调幅度（最近N天的最高价到当前收盘价的跌幅）
4. 检查回调幅度是否在合理范围内（3%-15%）
5. 检查成交量是否处于缩量状态（成交量比 < 1.0）
6. 检查企稳信号（价格不再创新低或开始回升）
7. 过滤 ST、*ST、退市股票
8. 过滤市值不符合的股票
9. 返回选股结果

### 性能指标

- 处理500只股票耗时 < 30秒
- 内存占用 < 500MB
- 支持实时更新

## 注意事项

### 适用场景

1. **上涨趋势中的回调**：缩量回调形态在上涨趋势中出现，成功率较高
2. **底部反转**：在长期下跌后出现缩量回调，可能预示反转
3. **突破整理平台**：突破横盘整理平台后回调，信号更可靠

### 不适用场景

1. **下跌趋势中**：在下跌趋势中出现缩量回调，成功率较低
2. **高位震荡**：在高位震荡中出现缩量回调，可能是下跌中继
3. **重大利空**：在重大利空消息影响下，缩量回调可能失效

### 风险控制

1. **设置止损**：建议在回调低点设置止损，止损幅度5%-8%
2. **分批建仓**：不要一次性满仓，可以分批建仓
3. **结合其他指标**：结合MACD、KDJ、均线等指标提高成功率
4. **注意成交量**：成交量是缩量回调形态的重要验证指标
5. **关注大盘**：大盘环境对缩量回调策略的成功率有重要影响
