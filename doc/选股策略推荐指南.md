# 选股策略推荐指南

## 策略概述

当前系统包含6个选股策略，每个策略都有不同的适用场景和特点。本文档将根据不同的市场环境和投资目标，推荐合适的策略组合。

## 策略分类

### 按市场环境分类

#### 1. 上升趋势市场
- **趋势加速拐点** 🚀
  - 适用场景：股票处于上升趋势，寻找加速上涨机会
  - 风险等级：中等
  - 推荐指数：⭐⭐⭐⭐

- **碗口反弹策略** 🍲
  - 适用场景：上升趋势中的回调买入机会
  - 风险等级：低
  - 推荐指数：⭐⭐⭐⭐⭐

- **多方炮策略** 💥
  - 适用场景：上涨趋势延续，多方发力
  - 风险等级：中等
  - 推荐指数：⭐⭐⭐⭐

#### 2. 下跌/震荡市场
- **底部趋势拐点** 📈
  - 适用场景：深度下跌后的反转机会
  - 风险等级：高（抄底）
  - 推荐指数：⭐⭐⭐

- **启明星策略** ⭐
  - 适用场景：底部反转信号
  - 风险等级：中高（抄底）
  - 推荐指数：⭐⭐⭐

#### 3. 任何市场环境
- **B1形态匹配** 📊
  - 适用场景：基于历史形态相似度匹配
  - 风险等级：中等
  - 推荐指数：⭐⭐⭐⭐

### 按投资风格分类

#### 1. 稳健型（低风险）
- **碗口反弹策略** 🍲
  - 优势：趋势跟踪，回调买入，风险较低
  - 适用：稳健投资者，追求稳定收益
  - 推荐组合：碗口反弹策略 + B1形态匹配

#### 2. 激进型（高风险高收益）
- **趋势加速拐点** 🚀 + **多方炮策略** 💥
  - 优势：捕捉加速上涨和延续上涨机会
  - 适用：激进投资者，追求高收益
  - 推荐组合：趋势加速拐点 + 多方炮策略

#### 3. 抄底型（高风险）
- **底部趋势拐点** 📈 + **启明星策略** ⭐
  - 优势：捕捉底部反转机会
  - 适用：有经验的投资者，能够承受高风险
  - 推荐组合：底部趋势拐点 + 启明星策略

## 推荐策略组合

### 组合1：稳健增长组合（推荐指数：⭐⭐⭐⭐⭐）

**适用场景**：上升趋势市场，追求稳健收益

**策略组合**：
1. **碗口反弹策略** 🍲（权重：50%）
   - 参数建议：
     - N: 4（成交量倍数）
     - M: 5（回溯天数）
     - CAP: 40（市值阈值）
     - J_VAL: 30（J值上限）
     - duokong_pct: 3%（多空线偏离）
     - short_pct: 2%（趋势线偏离）

2. **B1形态匹配** 📊（权重：50%）
   - 参数建议：
     - lookback_days: 25（回看天数）
     - min_similarity: 65%（最小相似度，提高要求）
     - top_n_results: 10（结果数量）

**选股逻辑**：OR（任一策略满足即可）

**预期效果**：
- 选股数量：中等（10-30只）
- 选股质量：高
- 风险等级：低

**优势**：
- 碗口反弹策略确保上升趋势
- B1形态匹配增加历史成功概率
- 风险可控，适合长期持有

---

### 组合2：趋势加速组合（推荐指数：⭐⭐⭐⭐）

**适用场景**：上升趋势明显，追求快速收益

**策略组合**：
1. **趋势加速拐点** 🚀（权重：60%）
   - 参数建议：
     - uptrend_lookback_days: 20（上升趋势天数）
     - price_increase_threshold: 0.08（涨幅阈值8%）
     - volume_ratio_threshold: 2.0（成交量倍数）
     - distance_threshold: 0.15（距离阈值15%）
     - lowest_point_lookback_days: 40（最低点回溯）

2. **多方炮策略** 💥（权重：40%）
   - 参数建议：
     - first_candle_rise: 0.03（第一根阳线涨幅3%）
     - second_candle_fallback_ratio: 0.5（第二根阴线回调比例）
     - third_candle_rise: 0.03（第三根阳线涨幅3%）
     - third_candle_breakthrough: true（突破前高）
     - third_volume_expand_ratio: 1.2（第三根阳线放量比例）
     - enable_ma_filter: true（启用均线过滤）
     - ma_period: 20（均线周期）

**选股逻辑**：OR（任一策略满足即可）

**预期效果**：
- 选股数量：中等偏少（5-20只）
- 选股质量：高
- 风险等级：中等

**优势**：
- 趋势加速拐点捕捉上涨加速信号
- 多方炮策略确认上涨延续
- 适合短线操作

---

### 组合3：底部反转组合（推荐指数：⭐⭐⭐）

**适用场景**：市场下跌，寻找抄底机会

**策略组合**：
1. **底部趋势拐点** 📈（权重：50%）
   - 参数建议：
     - decline_threshold: 0.45（下跌幅度45%）
     - lookback_days: 120（回溯天数）
     - price_increase_threshold: 0.08（涨幅阈值8%）
     - volume_ratio_threshold: 2.5（成交量倍数）
     - macd_divergence_days: 20（底背离时间窗口）

2. **启明星策略** ⭐（权重：50%）
   - 参数建议：
     - lookback_days: 10（回溯天数）
     - small_body_ratio: 0.3（小实体比例）
     - long_candle_ratio: 0.5（长阳线比例）
     - volume_ratio: 1.5（成交量比例）

**选股逻辑**：OR（任一策略满足即可）

**预期效果**：
- 选股数量：少（3-10只）
- 选股质量：中等
- 风险等级：高

**优势**：
- 底部趋势拐点确认深度下跌和底背离
- 启明星策略确认底部反转形态
- 适合有经验的投资者

### 组合4：全市场覆盖组合（推荐指数：⭐⭐⭐⭐⭐）

**适用场景**：市场环境不明，需要全面覆盖

**策略组合**：

2. **趋势加速拐点** 🚀（权重：25%）
   - 参数建议：使用默认参数

3. **多方炮策略** 💥（权重：20%）
   - 参数建议：使用默认参数

4. **底部趋势拐点** 📈（权重：15%）
   - 参数建议：使用默认参数

5. **启明星策略** ⭐（权重：10%）
   - 参数建议：使用默认参数

**选股逻辑**：OR（任一策略满足即可）

**预期效果**：
- 选股数量：多（20-50只）
- 选股质量：中等
- 风险等级：中等

- 覆盖所有市场环境

### 组合5：高质量精选组合（推荐指数：⭐⭐⭐⭐⭐）

**适用场景**：追求高质量选股，严格控制风险

**策略组合**：
1. **碗口反弹策略** 🍲（权重：40%）
   - 参数建议（保守）：
     - N: 5（提高成交量要求）
     - M: 3（缩短回溯天数）
     - CAP: 50（提高市值要求）
     - J_VAL: 25（降低J值要求）
     - duokong_pct: 2%（缩小多空线范围）
     - short_pct: 1.5%（缩小趋势线范围）

2. **趋势加速拐点** 🚀（权重：30%）
   - 参数建议（保守）：
     - uptrend_lookback_days: 15（缩短上升趋势天数）
     - price_increase_threshold: 0.10（提高涨幅要求）
     - volume_ratio_threshold: 2.5（提高成交量要求）
     - distance_threshold: 0.10（缩小距离范围）

3. **多方炮策略** 💥（权重：30%）
   - 参数建议（保守）：
     - first_candle_rise: 0.05（提高第一根阳线涨幅要求）
     - third_candle_rise: 0.05（提高第三根阳线涨幅要求）
     - third_candle_breakthrough: true（必须突破前高）
     - third_volume_expand_ratio: 1.5（提高放量要求）
     - enable_ma_filter: true（启用均线过滤）
     - enable_macd_filter: true（启用MACD过滤）

**选股逻辑**：AND（所有策略都满足才选股）

**预期效果**：
- 选股数量：很少（1-5只）
- 选股质量：极高
- 风险等级：低

**优势**：
- 多重确认，确保选股质量
- 严格过滤，降低风险
- 适合追求高质量选股的投资者

## 策略使用建议

### 1. 根据市场环境选择组合

| 市场环境 | 推荐组合 | 选股逻辑 |
|---------|---------|---------|
| 上升趋势明显 | 组合2（趋势加速组合） | OR |
| 上升趋势震荡 | 组合1（稳健增长组合） | OR |
| 下跌趋势 | 组合3（底部反转组合） | OR |
| 震荡不明 | 组合4（全市场覆盖组合） | OR |
| 追求高质量 | 组合5（高质量精选组合） | AND |

### 2. 根据投资风格选择组合

| 投资风格 | 推荐组合 | 风险等级 |
|---------|---------|---------|
| 稳健型 | 组合1（稳健增长组合） | 低 |
| 激进型 | 组合2（趋势加速组合） | 中等 |
| 抄底型 | 组合3（底部反转组合） | 高 |
| 平衡型 | 组合4（全市场覆盖组合） | 中等 |
| 精选型 | 组合5（高质量精选组合） | 低 |

### 3. 参数调整建议

#### 提高选股质量
- 提高成交量要求（N、volume_ratio_threshold）
- 提高涨幅要求（price_increase_threshold）
- 提高市值要求（CAP）
- 缩小位置范围（duokong_pct、short_pct）
- 启用趋势过滤（enable_ma_filter、enable_macd_filter）

#### 提高选股数量
- 降低成交量要求
- 降低涨幅要求
- 降低市值要求
- 扩大位置范围
- 延长回溯天数（lookback_days、M）

### 4. 风险控制建议

1. **设置止损点**
   - 碗口反弹策略：止损点设在多空线下方3-5%
   - 趋势加速拐点：止损点设在长阳线开盘价下方5%
   - 多方炮策略：止损点设在第二根阴线最低价下方3%
   - 底部反转策略：止损点设在起涨点下方5-8%

2. **分批建仓**
   - 不要一次性满仓
   - 建议分2-3次建仓
   - 观察走势逐步加仓

3. **控制仓位**
   - 单只股票仓位不超过总资金的20%
   - 同一策略选出的股票不超过3只
   - 总持仓股票不超过10只

4. **定期复盘**
   - 每周复盘选股结果
   - 分析成功和失败的案例
   - 根据市场变化调整策略参数

## 总结

### 最佳实践

1. **优先推荐**：组合1（稳健增长组合）
   - 适合大多数投资者
   - 风险可控，收益稳定
   - 选股质量高

2. **激进投资者**：组合2（趋势加速组合）
   - 适合短线操作
   - 收益潜力大
   - 需要严格止损

3. **市场下跌时**：组合3（底部反转组合）
   - 适合有经验的投资者
   - 抄底机会
   - 风险较高

4. **追求质量**：组合5（高质量精选组合）
   - 适合长期投资
   - 选股质量极高
   - 风险最低

### 注意事项

1. **不要过度优化**：参数调整要适度，避免过拟合
2. **结合基本面**：技术分析结合基本面分析
3. **控制风险**：设置止损，控制仓位
4. **持续学习**：不断复盘总结，提高选股能力
