# 项目来源说明

## 项目基础

本项目（KHunter）基于开源项目 **a-share-quant-selector** 扩展开发。

| 项目 | 信息 |

| **项目名称** | a-share-quant-selector |

| **许可证** | MIT License |
| **项目描述** | 基于Python + akshare的A股量化选股系统 |

本项目继承原项目的 **MIT License**，并在此基础上进行扩展开发。

## 扩展内容

### 新增功能

1. **11种选股策略**

   - 启明星策略（新增）
   - 底部趋势拐点（新增）
   - 趋势加速拐点（新增）
   - 多金叉共振（新增）
   - 多死叉共振（新增）
   - 多方炮策略（新增）
   - 缩量回调策略（新增）
   - 阻力位突破策略（新增）
   - W底策略（新增）
   - M头策略（新增）

2. **Web管理界面**
   - 选股历史查询
   - 交易模拟系统
   - 数据更新功能

3. **模拟交易系统**
   - 虚拟账户管理
   - 持仓管理
   - 交易记录
   - 收益计算

4. **系统优化**
   - 改进的数据管理机制
   - 智能更新策略
   - 参数锁定和追踪
   - 多策略交集分析

### 代码改进

- 重构了系统架构
- 提升了代码质量
- 增加了测试覆盖率
- 完善了文档
