# 选股结果链接功能测试

## 测试目标
验证选股结果中的股票代码和名称是否可点击，点击后是否能弹出K线图

## 测试环境
- 系统: Windows
- 浏览器: Chrome/Edge
- Web服务: Flask 3.0.3

## 测试用例

### TC-001: 验证选股结果中的链接显示
**步骤**:
1. 打开Web界面 http://127.0.0.1:5000
2. 点击顶部"执行选股"按钮
3. 等待选股完成
4. 查看选股结果页面

**预期结果**:
- ✓ 选股结果中的股票代码和名称显示为蓝色链接
- ✓ 链接有下划线或其他视觉提示
- ✓ 鼠标悬停时链接颜色变深

### TC-002: 验证点击链接打开K线图
**步骤**:
1. 在选股结果页面中点击任意股票链接
2. 观察是否弹出K线图弹窗

**预期结果**:
- ✓ 弹出股票详情弹窗
- ✓ 弹窗标题显示"股票详情: {股票代码}"
- ✓ 显示K线图表
- ✓ 显示股票基本信息（价格、K值、D值、J值等）

### TC-003: 验证K线图显示内容
**步骤**:
1. 点击股票链接打开K线图
2. 查看图表内容

**预期结果**:
- ✓ 显示收盘价曲线（蓝色）
- ✓ 显示KDJ指标（K、D、J三条线）
- ✓ 显示最新价格、最高价、最低价
- ✓ 显示成交量、市值等信息

### TC-004: 验证关闭弹窗
**步骤**:
1. 打开K线图弹窗
2. 点击关闭按钮或弹窗外部

**预期结果**:
- ✓ 弹窗关闭
- ✓ 返回选股结果页面
- ✓ 可以继续点击其他股票链接

### TC-005: 验证多个股票链接
**步骤**:
1. 在选股结果中依次点击多个股票链接
2. 验证每个链接都能正常打开

**预期结果**:
- ✓ 所有股票链接都能正常打开K线图
- ✓ 每个K线图显示对应股票的数据
- ✓ 没有错误或异常

## 实现细节

### 前端修改
- 修改 `renderSelectionResults()` 函数
- 将股票代码和名称包装为可点击链接
- 链接调用 `viewStockDetail(code)` 函数

### CSS样式
- 添加 `.stock-link` 类
- 链接颜色: 主色（蓝色）
- 悬停效果: 颜色变深，显示下划线

### 用户体验
- 点击链接后立即弹出K线图
- 支持关闭弹窗后继续点击其他链接
- 无需刷新页面
